# واجهة برمجة تطبيقات REST العامة لـ n8n

/// info | توفر الميزة
واجهة برمجة تطبيقات n8n غير متاحة خلال الفترة التجريبية المجانية. يرجى الترقية للوصول إلى هذه الميزة.
///

باستخدام [واجهة برمجة التطبيقات](/glossary.md#api) العامة لـ n8n، يمكنك أداء العديد من المهام برمجياً بنفس الطريقة التي تؤديها في الواجهة الرسومية. يقدم هذا القسم واجهة برمجة تطبيقات REST الخاصة بـ n8n، بما في ذلك:

*   كيفية [المصادقة](/api/authentication.md)
*   [تقسيم](/api/pagination.md) النتائج إلى صفحات
*   استخدام [ملعب واجهة برمجة التطبيقات المدمج](/api/using-api-playground.md) (لنسخ n8n المستضافة ذاتياً فقط)
*   [مرجع نقاط النهاية](/api/api-reference.md)

توفر n8n [عقدة n8n API](/integrations/builtin/core-nodes/n8n-nodes-base.n8n.md) للوصول إلى واجهة برمجة التطبيقات في سير العمل الخاص بك.

## اختر طريقة التفاعل الخاصة بك

### واجهة برمجة تطبيقات REST (هذا القسم)
تفاعل مع n8n مباشرة باستخدام طلبات HTTP. مثالي لـ:
- التكاملات والتطبيقات المخصصة
- استدعاءات HTTP المستقلة عن اللغة
- الاستخدام المباشر لواجهة برمجة تطبيقات REST في سير العمل

### واجهة سطر الأوامر لـ n8n (موصى بها للمطورين)
استخدم [واجهة سطر الأوامر لـ n8n](/api/n8n-cli/index.md) لتجربة سطر الأوامر. إنها تغلف واجهة برمجة التطبيقات العامة ومُحسّنة لـ:
- أتمتة سطر الأوامر والبرمجة النصية
- تكامل مسار CI/CD
- تكامل وكيل الذكاء الاصطناعي (Claude Code، Cursor، إلخ.)
- المطورون الذين يفضلون أدوات سطر الأوامر

## تعرف على واجهات برمجة تطبيقات REST

يفترض توثيق واجهة برمجة التطبيقات أنك على دراية بواجهات برمجة تطبيقات REST. إذا لم تكن كذلك، فقد تكون هذه الموارد مفيدة:

*   [دليل KnowledgeOwl للعمل مع واجهات برمجة التطبيقات](https://support.knowledgeowl.com/help/working-with-apis): مقدمة أساسية، تتضمن أمثلة لكيفية استدعاء واجهات برمجة تطبيقات REST.
*   [مركز تعلم IBM Cloud - ما هي واجهة برمجة التطبيقات (API)](https://www.ibm.com/cloud/learn/api): يقدم هذا مقدمة عامة، ولكنها تقنية، لواجهات برمجة التطبيقات.
*   [مركز تعلم IBM Cloud - ما هي واجهة برمجة تطبيقات REST؟](https://www.ibm.com/cloud/learn/rest-apis): معلومات أكثر تفصيلاً حول واجهات برمجة تطبيقات REST.
*   [وثائق الويب MDN - نظرة عامة على HTTP](https://developer.mozilla.org/en-US/docs/Web/HTTP/Overview): تعمل واجهات برمجة تطبيقات REST عبر HTTP وتستخدم أفعال HTTP، أو طرقها، لتحديد الإجراء المراد تنفيذه.

/// tip | استخدم ملعب واجهة برمجة التطبيقات
يمكن أن يساعدك تجربة واجهة برمجة التطبيقات في [الملعب](/api/using-api-playground.md) على فهم كيفية عمل واجهات برمجة التطبيقات. إذا كنت قلقاً بشأن تغيير البيانات الحية، ففكر في إعداد سير عمل اختباري، أو نسخة n8n اختبارية، للاستكشاف بأمان.
