# مصادقة API

تستخدم n8n مفاتيح API للمصادقة على استدعاءات API.

/// info | توفر الميزة
واجهة برمجة تطبيقات n8n غير متاحة خلال الفترة التجريبية المجانية. يرجى الترقية للوصول إلى هذه الميزة.
///

## نطاقات API

يمكن لمستخدمي [نسخ المؤسسات](https://n8n.io/enterprise/) تحديد الموارد والإجراءات التي يمكن للمفتاح الوصول إليها باستخدام النطاقات. تسمح نطاقات مفتاح API بتحديد مستوى الوصول الدقيق الذي يحتاجه المفتاح لغرضه المقصود.

تتمتع مفاتيح API غير الخاصة بالمؤسسات بوصول كامل إلى جميع موارد الحساب وقدراته.

## إنشاء مفتاح API

1. سجّل الدخول إلى n8n.
2. انتقل إلى **الإعدادات** > **n8n API**.
3. اختر **إنشاء مفتاح API**.
4. اختر **تسمية** وحدد وقت **انتهاء الصلاحية** للمفتاح.
5. إذا كنت تستخدم خطة مؤسسية، فاختر **النطاقات** التي ستمنحها للمفتاح.
6. انسخ **مفتاح API الخاص بي** واستخدم هذا المفتاح للمصادقة على استدعاءاتك.

## استدعاء API باستخدام مفتاحك

أرسل مفتاح API في استدعاء API الخاص بك كرأس (header) باسم `X-N8N-API-KEY`.

على سبيل المثال، إذا كنت ترغب في الحصول على جميع سير العمل النشطة. سيبدو طلب curl الخاص بك كالتالي:

```shell
# For a self-hosted n8n instance
curl -X 'GET' \
  '<N8N_HOST>:<N8N_PORT>/<N8N_PATH>/api/v<version-number>/workflows?active=true' \
  -H 'accept: application/json' \
  -H 'X-N8N-API-KEY: <your-api-key>'

# For n8n Cloud
curl -X 'GET' \
  '<your-cloud-instance>/api/v<version-number>/workflows?active=true' \
  -H 'accept: application/json' \
  -H 'X-N8N-API-KEY: <your-api-key>'
```

## حذف مفتاح API

1. سجّل الدخول إلى n8n.
2. انتقل إلى **الإعدادات** > **n8n API**.
3. اختر **حذف** بجانب المفتاح الذي تريد حذفه.
4. أكّد الحذف باختيار **حذف نهائي**.
