# الأساليب والمتغيرات المضمنة

توفر n8n أساليب ومتغيرات مضمنة للتعامل مع البيانات والوصول إلى بيانات n8n. يقدم هذا القسم مرجعًا للأساليب والمتغيرات المتاحة للاستخدام في [التعبيرات](/glossary.md#expression-n8n)، مع وصف موجز.

/// ملاحظة | التوفر في محرر التعبيرات وعقدة Code
بعض الأساليب والمتغيرات غير متاحة في عقدة Code. هذه غير موجودة في التوثيق.

جميع وظائف تحويل البيانات متاحة فقط في محرر التعبيرات.
///
