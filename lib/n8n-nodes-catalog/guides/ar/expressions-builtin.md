# الأساليب والمتغيرات المدمجة

توفر n8n أساليب ومتغيرات مدمجة للتعامل مع البيانات والوصول إلى بيانات n8n. يقدم هذا القسم مرجعاً للأساليب والمتغيرات المتاحة للاستخدام في [التعبيرات](/glossary.md#expression-n8n)، مع وصف موجز.

/// ملاحظة | التوفر في محرر التعبيرات وعقدة Code
بعض الأساليب والمتغيرات غير متاحة في عقدة Code (الكود). هذه غير موجودة في التوثيق.

جميع وظائف تحويل البيانات متاحة فقط في محرر التعبيرات.
///

يحتوي [دليل الوصفات](/code/index.md) على أمثلة لبعض المهام الشائعة، بما في ذلك بعض وظائف [عقدة Code فقط](/code/cookbook/code-node/index.md).
