# الطرق والمتغيرات المدمجة

يُوفِّر n8n طرقاً ومتغيرات مدمجة للعمل مع البيانات والوصول إلى بيانات n8n. يقدم هذا القسم مرجعاً للطرق والمتغيرات المتاحة للاستخدام في [التعبيرات](/glossary.md#expression-n8n)، مع وصف موجز.

/// ملاحظة | التوفر في محرر التعبيرات والعقدة Code (عقدة الكود)
بعض الطرق والمتغيرات غير متاحة في العقدة Code (عقدة الكود). وليست مدرجة في التوثيق.
جميع دوال تحويل البيانات متاحة فقط في محرر التعبيرات.
///		

يحتوي [Cookbook](/code/index.md) على أمثلة لبعض المهام الشائعة، بما في ذلك بعض وظائف [Code node only (عقدة الكود فقط)](/code/cookbook/code-node/index.md).
