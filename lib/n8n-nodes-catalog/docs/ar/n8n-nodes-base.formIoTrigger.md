# العقدة المُحفِّزة Form.io

Form.io (فورم.آي.أو) هي منصة مدمجة لإدارة النماذج وبيانات API من فئة المؤسسات، مخصصة لبناء تطبيقات معقدة لعمليات الأعمال القائمة على النماذج.

> **بيانات الاعتماد**
>
> يمكن العثور على معلومات المصادقة لهذه العقدة [هنا](/integrations/builtin/credentials/formiotrigger.md).

> **أمثلة وقوالب**
>
> للحصول على أمثلة الاستخدام والقوالب لمساعدتك على البدء، ارجع إلى صفحة [تكاملات Form.io Trigger](https://n8n.io/integrations/formio-trigger/) الخاصة بـ n8n.
