# عقدة Disqus

استخدم عقدة Disqus لأتمتة سير العمل في Disqus، ودمج Disqus مع تطبيقات أخرى. تدعم n8n بشكل مدمج مجموعة واسعة من ميزات Disqus، بما في ذلك استرجاع المنتديات.

في هذه الصفحة، ستجد قائمة بالعمليات التي تدعمها عقدة Disqus وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد Disqus](/integrations/builtin/credentials/disqus.md) للحصول على إرشادات حول إعداد المصادقة.

## العمليات

* المنتدى
    * استرجاع تفاصيل المنتدى
    * استرجاع قائمة بالفئات ضمن منتدى
    * استرجاع قائمة بالمواضيع ضمن منتدى
    * استرجاع قائمة بالمنشورات ضمن منتدى
