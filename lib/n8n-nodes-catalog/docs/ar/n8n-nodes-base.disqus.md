# عقدة Disqus

استخدم عقدة Disqus لأتمتة سير العمل في Disqus، ودمج Disqus مع تطبيقات أخرى. تدعم n8n مجموعة واسعة من ميزات Disqus بشكل مدمج، بما في ذلك استرجاع المنتديات.

ستجد في هذه الصفحة قائمة بالعمليات التي تدعمها عقدة Disqus وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد Disqus](/integrations/builtin/credentials/disqus.md) للحصول على إرشادات حول إعداد المصادقة.

## العمليات

*   المنتدى
    *   استرجاع تفاصيل المنتدى
    *   استرجاع قائمة بالفئات داخل منتدى
    *   استرجاع قائمة بالمواضيع داخل منتدى
    *   استرجاع قائمة بالمنشورات داخل منتدى
