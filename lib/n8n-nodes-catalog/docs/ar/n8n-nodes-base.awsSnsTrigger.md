# عقدة مُحفِّز AWS SNS

[AWS SNS](https://aws.amazon.com/sns/) (خدمة الإشعارات البسيطة من أمازون) هي خدمة إشعارات تُقدّم كجزء من خدمات الويب من أمازون. توفر بنية تحتية منخفضة التكلفة لتسليم الرسائل بكميات كبيرة، وبشكل أساسي للمستخدمين المتنقلين.

> **بيانات الاعتماد**
>
> يمكنك العثور على معلومات المصادقة لهذه العقدة [هنا](/integrations/builtin/credentials/aws.md).

> **أمثلة وقوالب**
>
> للحصول على أمثلة الاستخدام والقوالب لمساعدتك على البدء، ارجع إلى صفحة [تكاملات مُحفِّز AWS SNS](https://n8n.io/integrations/aws-sns-trigger/) الخاصة بـ n8n.

## الأحداث

*   حدث AWS SNS جديد

## الموارد ذات الصلة

توفر n8n عقدة تطبيق لـ AWS SNS. يمكنك العثور على وثائق العقدة [هنا](/integrations/builtin/app-nodes/n8n-nodes-base.awssns.md).

اطلع على [أمثلة سير العمل والمحتوى ذي الصلة](https://n8n.io/integrations/aws-sns-trigger/) على موقع n8n الإلكتروني.

ارجع إلى [وثائق AWS SNS](https://docs.aws.amazon.com/sns/latest/api/welcome.html) للحصول على تفاصيل حول واجهة برمجة التطبيقات (API) الخاصة بهم.
