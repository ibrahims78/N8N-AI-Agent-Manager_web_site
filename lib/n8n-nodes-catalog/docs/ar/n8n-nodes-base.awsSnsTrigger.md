# عقدة مُحفِّز AWS SNS

[خدمة الإشعارات البسيطة من أمازون ويب سيرفيسز (AWS SNS)](https://aws.amazon.com/sns/) هي خدمة إشعارات تُقدَّم كجزء من خدمات الويب من أمازون. توفر بنية تحتية منخفضة التكلفة للتسليم الجماعي للرسائل، بشكل أساسي لمستخدمي الهواتف المحمولة.

> **بيانات الاعتماد**
>
> يمكن العثور على معلومات المصادقة لهذه العقدة [هنا](/integrations/builtin/credentials/aws.md).

> **أمثلة وقوالب**
>
> لأمثلة الاستخدام والقوالب التي تساعدك على البدء، ارجع إلى صفحة [عمليات تكامل مُحفِّز AWS SNS](https://n8n.io/integrations/aws-sns-trigger/) الخاصة بـ n8n.

## الأحداث

*   حدث AWS SNS جديد

## الموارد ذات الصلة

توفر n8n عقدة تطبيق لـ AWS SNS. يمكنك العثور على وثائق العقدة [هنا](/integrations/builtin/app-nodes/n8n-nodes-base.awssns.md).

اطلع على [أمثلة سير العمل والمحتوى ذي الصلة](https://n8n.io/integrations/aws-sns-trigger/) على موقع n8n الإلكتروني.

ارجع إلى [وثائق AWS SNS](https://docs.aws.amazon.com/sns/latest/api/welcome.html) للحصول على تفاصيل حول واجهة برمجة التطبيقات الخاصة بهم.
