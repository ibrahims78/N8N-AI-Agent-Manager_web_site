# عقدة Airtop

استخدم عقدة Airtop لأتمتة سير العمل في Airtop، ودمج Airtop مع تطبيقات أخرى. تدعم n8n مجموعة واسعة من ميزات Airtop بشكل مدمج، مما يتيح لك التحكم في متصفح ويب قائم على السحابة لمهام مثل الاستعلام، وكشط البيانات، والتفاعل مع صفحات الويب.

ستجد في هذه الصفحة قائمة بالعمليات التي تدعمها عقدة Airtop، وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد Airtop](/integrations/builtin/credentials/airtop.md) للحصول على إرشادات حول إعداد المصادقة.

## العمليات

*   الجلسة
    *   إنشاء جلسة
    *   حفظ الملف الشخصي عند الإنهاء
    *   إنهاء الجلسة
*   النافذة
    *   إنشاء نافذة متصفح جديدة
    *   تحميل URL
    *   التقاط لقطة شاشة
    *   إغلاق النافذة
*   الاستخراج
    *   استعلام الصفحة
    *   استعلام الصفحة مع ترقيم الصفحات
    *   كشط الصفحة بذكاء
*   التفاعل
    *   النقر على عنصر
    *   التحويم فوق عنصر
    *   الكتابة

## الموارد ذات الصلة

ارجع إلى [توثيق Airtop](https://docs.airtop.ai/api-reference/airtop-api) لمزيد من المعلومات حول الخدمة.

## ماذا تفعل إذا كانت عمليتك غير مدعومة

إذا كانت هذه العقدة لا تدعم العملية التي ترغب في تنفيذها، يمكنك استخدام [عقدة HTTP Request (طلب HTTP)](/integrations/builtin/core-nodes/n8n-nodes-base.httprequest/index.md) لاستدعاء واجهة برمجة تطبيقات (API) الخدمة.

يمكنك استخدام بيانات الاعتماد التي أنشأتها لهذه الخدمة في عقدة HTTP Request:

1.  في عقدة HTTP Request، حدد **المصادقة** > **نوع بيانات الاعتماد المُحددة مسبقًا**.
2.  حدد الخدمة التي ترغب في الاتصال بها.
3.  حدد بيانات الاعتماد الخاصة بك.

ارجع إلى [عمليات API المخصصة](/integrations/custom-operations.md) لمزيد من المعلومات.

تواصل مع [دعم Airtop](https://docs.airtop.ai/guides/misc/support) للحصول على المساعدة أو لإنشاء طلب ميزة.

## مرجع العقدة

### إنشاء جلسة ونافذة

أنشئ جلسة متصفح Airtop للحصول على **معرف الجلسة (Session ID)**، ثم استخدمه لإنشاء نافذة متصفح جديدة. بعد ذلك، يمكنك استخدام أي عملية استخراج أو تفاعل.

### استخراج المحتوى

استخرج المحتوى من متصفح الويب باستخدام هذه العمليات:

*   **استعلام الصفحة**: استخراج المعلومات من النافذة الحالية.
*   **استعلام الصفحة مع ترقيم الصفحات**: استخراج المعلومات من الصفحات ذات ترقيم الصفحات أو التمرير اللانهائي.
*   **كشط الصفحة بذكاء**: الحصول على محتوى النافذة بصيغة Markdown.

احصل على استجابات JSON باستخدام المعامل **مخطط مخرجات JSON (JSON Output Schema)** في عمليات الاستعلام.

### التفاعل مع الصفحات

انقر، أو حوّم، أو اكتب على العناصر عن طريق وصف العنصر الذي ترغب في التفاعل معه.

### إنهاء جلسة

أنهِ جلستك لتوفير الموارد. يتم إنهاء الجلسات تلقائيًا بناءً على **مهلة الخمول (Idle Timeout)** المحددة في عملية **إنشاء جلسة (Create Session)** أو يمكن إنهاؤها يدويًا باستخدام عملية **إنهاء جلسة (Terminate Session)**.
