# عقدة HighLevel

استخدم عقدة HighLevel لأتمتة سير العمل في HighLevel، ودمج HighLevel مع تطبيقات أخرى. تدعم n8n بشكل مدمج مجموعة واسعة من ميزات HighLevel، بما في ذلك إنشاء وتحديث وحذف والحصول على جهات الاتصال والفرص والمهام، بالإضافة إلى حجز المواعيد والحصول على الفترات الزمنية المتاحة في التقويمات.

في هذه الصفحة، ستجد قائمة بالعمليات التي تدعمها عقدة HighLevel وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد HighLevel](/integrations/builtin/credentials/highlevel.md) للحصول على إرشادات حول إعداد المصادقة.

## الموارد ذات الصلة

ارجع إلى [وثائق API ومنتديات دعم HighLevel](https://help.gohighlevel.com/support/solutions/articles/48001060529-highlevel-api) لمزيد من المعلومات حول الخدمة.

## ماذا تفعل إذا كانت عمليتك غير مدعومة

إذا كانت هذه العقدة لا تدعم العملية التي ترغب في تنفيذها، يمكنك استخدام [عقدة HTTP Request](/integrations/builtin/core-nodes/n8n-nodes-base.httprequest/index.md) لاستدعاء API الخاص بالخدمة.

يمكنك استخدام بيانات الاعتماد التي أنشأتها لهذه الخدمة في عقدة HTTP Request:

1.  في عقدة HTTP Request، حدد **المصادقة** > **نوع بيانات الاعتماد المُحددة مسبقًا**.
2.  حدد الخدمة التي ترغب في الاتصال بها.
3.  حدد بيانات الاعتماد الخاصة بك.

ارجع إلى [عمليات API المخصصة](/integrations/custom-operations.md) لمزيد من المعلومات.
