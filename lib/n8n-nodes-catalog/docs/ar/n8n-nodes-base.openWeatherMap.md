# عقدة OpenWeatherMap

استخدم عقدة OpenWeatherMap لأتمتة سير العمل في OpenWeatherMap، ودمج OpenWeatherMap مع تطبيقات أخرى. تدعم n8n استرجاع بيانات الطقس الحالية والقادمة باستخدام OpenWeatherMap.

في هذه الصفحة، ستجد قائمة بالعمليات التي تدعمها عقدة OpenWeatherMap وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد OpenWeatherMap](/integrations/builtin/credentials/openweathermap.md) للحصول على إرشادات حول إعداد المصادقة.

> **يمكن استخدام هذه العقدة كأداة للذكاء الاصطناعي**
>
> يمكن استخدام هذه العقدة لتعزيز قدرات وكيل الذكاء الاصطناعي. عند استخدامها بهذه الطريقة، يمكن تعيين العديد من المعاملات تلقائيًا، أو بمعلومات موجهة بواسطة الذكاء الاصطناعي - اكتشف المزيد في [توثيق معاملات أداة الذكاء الاصطناعي](/advanced-ai/examples/using-the-fromai-function.md).

## العمليات

*   تُرجع بيانات الطقس الحالية
*   تُرجع بيانات الطقس للأيام الخمسة القادمة
