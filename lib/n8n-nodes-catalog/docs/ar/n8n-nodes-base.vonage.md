# عقدة Vonage

استخدم عقدة Vonage لأتمتة سير العمل في Vonage، ودمج Vonage مع تطبيقات أخرى. تدعم n8n إرسال الرسائل القصيرة (SMS) باستخدام Vonage.

في هذه الصفحة، ستجد قائمة بالعمليات التي تدعمها عقدة Vonage وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد Vonage](/integrations/builtin/credentials/vonage.md) للحصول على إرشادات حول إعداد المصادقة.

## العمليات

    * إرسال
