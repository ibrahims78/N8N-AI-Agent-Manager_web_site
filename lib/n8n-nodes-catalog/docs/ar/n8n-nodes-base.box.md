# عقدة Box (خدمة تخزين سحابي)

استخدم عقدة Box لأتمتة سير العمل في Box، ودمج Box مع تطبيقات أخرى. تدعم n8n بشكل مدمج مجموعة واسعة من ميزات Box، بما في ذلك إنشاء الملفات والمجلدات ونسخها وحذفها والبحث عنها ورفعها وتنزيلها.

في هذه الصفحة، ستجد قائمة بالعمليات التي تدعمها عقدة Box وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد Box](/integrations/builtin/credentials/box.md) للحصول على إرشادات حول إعداد المصادقة.
 

## العمليات

* الملف
    * نسخ ملف
    * حذف ملف
    * تنزيل ملف
    * الحصول على ملف
    * البحث عن ملفات
    * مشاركة ملف
    * رفع ملف
* المجلد
    * إنشاء مجلد
    * الحصول على مجلد
    * حذف مجلد
    * البحث عن ملفات
    * مشاركة مجلد
    * تحديث مجلد

## ماذا تفعل إذا كانت عمليتك غير مدعومة

إذا كانت هذه العقدة لا تدعم العملية التي ترغب في تنفيذها، يمكنك استخدام [عقدة HTTP Request](/integrations/builtin/core-nodes/n8n-nodes-base.httprequest/index.md) لاستدعاء واجهة برمجة تطبيقات (API) الخدمة.

يمكنك استخدام بيانات الاعتماد التي أنشأتها لهذه الخدمة في عقدة HTTP Request: 

1. في عقدة HTTP Request، حدد **المصادقة** > **نوع بيانات الاعتماد المُحددة مسبقًا**.
2. حدد الخدمة التي ترغب في الاتصال بها.
3. حدد بيانات الاعتماد الخاصة بك.

ارجع إلى [عمليات API المخصصة](/integrations/custom-operations.md) لمزيد من المعلومات.
