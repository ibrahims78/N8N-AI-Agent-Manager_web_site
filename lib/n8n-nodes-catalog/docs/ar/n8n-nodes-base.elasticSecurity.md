# عقدة Elastic Security

استخدم عقدة Elastic Security (أمان Elastic) لأتمتة سير العمل في Elastic Security (أمان Elastic)، ودمج Elastic Security مع تطبيقات أخرى. تدعم n8n مجموعة واسعة من ميزات Elastic Security بشكل مدمج، بما في ذلك إنشاء الحالات وتحديثها وحذفها واستردادها والحصول عليها.

ستجد في هذه الصفحة قائمة بالعمليات التي تدعمها عقدة Elastic Security وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد Elastic Security](/integrations/builtin/credentials/elasticsecurity.md) للحصول على إرشادات حول إعداد المصادقة.

## العمليات

*   الحالة
    *   إنشاء حالة
    *   حذف حالة
    *   الحصول على حالة
    *   استرداد جميع الحالات
    *   استرداد ملخص لجميع أنشطة الحالات
    *   تحديث حالة
*   تعليق الحالة
    *   إضافة تعليق إلى حالة
    *   الحصول على تعليق حالة
    *   استرداد جميع تعليقات الحالات
    *   إزالة تعليق من حالة
    *   تحديث تعليق في حالة
*   وسم الحالة
    *   إضافة وسم إلى حالة
    *   إزالة وسم من حالة
*   الموصل
    *   إنشاء موصل
