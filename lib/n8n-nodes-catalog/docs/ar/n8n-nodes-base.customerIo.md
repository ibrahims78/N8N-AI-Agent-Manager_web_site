# عقدة Customer.io

استخدم عقدة Customer.io لأتمتة سير العمل في Customer.io، ودمج Customer.io مع تطبيقات أخرى. تدعم n8n بشكل مدمج مجموعة واسعة من ميزات Customer.io، بما في ذلك إنشاء العملاء وتحديثهم، وتتبع الأحداث، والحصول على الحملات.

ستجد في هذه الصفحة قائمة بالعمليات التي تدعمها عقدة Customer.io وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد Customer.io](/integrations/builtin/credentials/customerio.md) للحصول على إرشادات حول إعداد المصادقة.

## العمليات

*   العميل
    *   إنشاء/تحديث عميل.
    *   حذف عميل.
*   الحدث
    *   تتبع حدث عميل.
    *   تتبع حدث مجهول.
*   الحملة
    *   الحصول
    *   الحصول على الكل
    *   الحصول على المقاييس
*   الشريحة
    *   إضافة عميل
    *   إزالة عميل

## ماذا تفعل إذا كانت عمليتك غير مدعومة

إذا كانت هذه العقدة لا تدعم العملية التي ترغب في تنفيذها، يمكنك استخدام [عقدة HTTP Request](/integrations/builtin/core-nodes/n8n-nodes-base.httprequest/index.md) لاستدعاء واجهة برمجة تطبيقات الخدمة.

يمكنك استخدام بيانات الاعتماد التي أنشأتها لهذه الخدمة في عقدة HTTP Request:

1.  في عقدة HTTP Request، حدد **المصادقة** > **نوع بيانات الاعتماد المُحددة مسبقًا**.
2.  حدد الخدمة التي ترغب في الاتصال بها.
3.  حدد بيانات الاعتماد الخاصة بك.

ارجع إلى [عمليات API المخصصة](/integrations/custom-operations.md) لمزيد من المعلومات.
