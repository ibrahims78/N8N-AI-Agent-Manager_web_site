# عقدة مُحفِّز Microsoft Outlook

استخدم عقدة مُحفِّز Microsoft Outlook للاستجابة للأحداث في [Microsoft Outlook](https://www.microsoft.com/en-us/microsoft-365/outlook/email-and-calendar-software-microsoft-outlook) ودمج Microsoft Outlook مع التطبيقات الأخرى.

في هذه الصفحة، ستجد قائمة بالأحداث التي يمكن لعقدة مُحفِّز Microsoft Outlook الاستجابة لها، وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> يمكنك العثور على معلومات المصادقة لهذه العقدة [هنا](/integrations/builtin/credentials/microsoft.md).

> **دعم السحابة الحكومية**
>
> إذا كنت تستخدم مستأجر سحابة حكومية (حكومة الولايات المتحدة، وزارة دفاع حكومة الولايات المتحدة، أو الصين)، تأكد من تحديد **عنوان URL الأساسي لـ Microsoft Graph API** المناسب في إعدادات بيانات اعتماد Microsoft الخاصة بك.

> **أمثلة وقوالب**
>
> للحصول على أمثلة الاستخدام والقوالب لمساعدتك على البدء، ارجع إلى صفحة [تكاملات Microsoft Outlook](https://n8n.io/integrations/microsoft-outlook-trigger/) الخاصة بـ n8n.

## الأحداث

*   استلام رسالة

## الموارد ذات الصلة

توفر n8n عقدة تطبيق لـ Microsoft Outlook. يمكنك العثور على وثائق العقدة [هنا](/integrations/builtin/app-nodes/n8n-nodes-base.microsoftoutlook.md).

عرض [أمثلة سير العمل والمحتوى ذي الصلة](https://n8n.io/integrations/microsoft-outlook-trigger/) على موقع n8n الإلكتروني.
