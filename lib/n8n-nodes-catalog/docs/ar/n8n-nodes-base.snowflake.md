# عقدة Snowflake

استخدم عقدة Snowflake لأتمتة سير العمل في Snowflake، ودمج Snowflake مع تطبيقات أخرى. تدعم n8n بشكل مدمج مجموعة واسعة من ميزات Snowflake، بما في ذلك تنفيذ استعلامات SQL، وإدراج صفوف في قاعدة بيانات.

في هذه الصفحة، ستجد قائمة بالعمليات التي تدعمها عقدة Snowflake وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد Snowflake](/integrations/builtin/credentials/snowflake.md) للحصول على إرشادات حول إعداد المصادقة.

## العمليات

*   تنفيذ استعلام SQL.
*   إدراج صفوف في قاعدة البيانات.
*   تحديث صفوف في قاعدة البيانات.
