# عقدة Redis

استخدم عقدة Redis لأتمتة سير العمل في Redis، ودمج Redis مع تطبيقات أخرى. تدعم n8n بشكل مدمج مجموعة واسعة من ميزات Redis، بما في ذلك حذف المفاتيح، والحصول على قيم المفاتيح، وتعيين قيمة المفتاح، ونشر الرسائل إلى قناة Redis.

في هذه الصفحة، ستجد قائمة بالعمليات التي تدعمها عقدة Redis وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد Redis](/integrations/builtin/credentials/redis.md) للحصول على إرشادات حول إعداد المصادقة.

> **يمكن استخدام هذه العقدة كأداة ذكاء اصطناعي**
>
> يمكن استخدام هذه العقدة لتعزيز قدرات وكيل الذكاء الاصطناعي. عند استخدامها بهذه الطريقة، يمكن تعيين العديد من المعاملات تلقائيًا، أو بمعلومات موجهة بواسطة الذكاء الاصطناعي - اكتشف المزيد في [توثيق معاملات أداة الذكاء الاصطناعي](/advanced-ai/examples/using-the-fromai-function.md).

## العمليات

*   حذف مفتاح من Redis.
*   الحصول على قيمة مفتاح من Redis.
*   إرجاع معلومات عامة حول نسخة Redis.
*   زيادة مفتاح بمقدار 1 بشكل ذري. تنشئ المفتاح إذا لم يكن موجودًا.
*   إرجاع جميع المفاتيح المطابقة لنمط معين.
*   تعيين قيمة مفتاح في Redis.
*   نشر رسالة إلى قناة Redis.
