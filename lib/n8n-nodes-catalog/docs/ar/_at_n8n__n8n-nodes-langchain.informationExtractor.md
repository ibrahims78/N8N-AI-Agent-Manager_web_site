# عقدة استخراج المعلومات (Information Extractor)

استخدم عقدة استخراج المعلومات لاستخراج معلومات منظمة من البيانات الواردة.

في هذه الصفحة، ستجد معاملات العقدة لعقدة استخراج المعلومات، وروابط لموارد إضافية.

## معاملات العقدة

*   **Text** يُعرّف نص الإدخال الذي ستُستخرج منه المعلومات. عادةً ما يكون هذا تعبيراً يشير إلى حقل من عناصر الإدخال. على سبيل المثال، قد يكون `` إذا كان الإدخال مُحفِّز دردشة، أو `` إذا كانت العقدة السابقة هي Extract from PDF.
*   استخدم **Schema Type** لاختيار كيفية وصف تنسيق بيانات المخرجات المطلوب. يمكنك الاختيار بين:
    *   **From Attribute Descriptions**: يتيح لك هذا الخيار تعريف المخطط عن طريق تحديد قائمة السمات وأوصافها.
    *   **Generate From JSON Example**: أدخل كائن JSON مثالياً لتوليد المخطط تلقائياً. تستخدم العقدة أنواع وأسماء خصائص الكائن. وتتجاهل القيم الفعلية. تتعامل n8n مع كل حقل على أنه مطلوب عند توليد المخططات من أمثلة JSON.
    *   **Define using JSON Schema**: أدخل مخطط JSON يدوياً. اقرأ [الأدلة والأمثلة](https://json-schema.org/learn/miscellaneous-examples) الخاصة بمخطط JSON للمساعدة في إنشاء مخطط JSON صالح.

## خيارات العقدة

*   **System Prompt Template**: استخدم هذا الخيار لتغيير موجه النظام المُستخدم لاستخراج المعلومات. تُلحق n8n تلقائياً تعليمات تحديد التنسيق بالموجه.
