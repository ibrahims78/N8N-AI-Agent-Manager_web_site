# عقدة مُحفِّز Customer.io

تُمكّن [Customer.io](https://customer.io/) (منصة لأتمتة التسويق وإدارة علاقات العملاء) المستخدمين من إرسال الرسائل الإخبارية إلى شرائح مختارة من العملاء باستخدام بيانات موقعهم الإلكتروني. يمكنك إرسال رسائل بريد إلكتروني مستهدفة، وإشعارات دفع، ورسائل نصية قصيرة (SMS) لتقليل معدل التوقف عن الخدمة، وبناء علاقات أقوى، وزيادة الاشتراكات.

> **بيانات الاعتماد**
>
> يمكنك العثور على معلومات المصادقة لهذه العقدة [هنا](/integrations/builtin/credentials/customerio.md).

> **أمثلة وقوالب**
>
> للحصول على أمثلة الاستخدام والقوالب لمساعدتك على البدء، ارجع إلى صفحة [تكاملات Customer.io Trigger](https://n8n.io/integrations/customerio-trigger/) الخاصة بـ n8n.

## الموارد ذات الصلة

توفر n8n عقدة تطبيق لـ Customer.io. يمكنك العثور على وثائق العقدة [هنا](/integrations/builtin/app-nodes/n8n-nodes-base.customerio.md).

اعرض [أمثلة سير العمل والمحتوى ذي الصلة](https://n8n.io/integrations/customerio-trigger/) على موقع n8n الإلكتروني.

ارجع إلى [وثائق Customer.io](https://docs.customer.io/api/) للحصول على تفاصيل حول API الخاص بهم.
