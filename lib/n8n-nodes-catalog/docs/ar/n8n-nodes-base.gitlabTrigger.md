# العقدة المُحفِّزة لـ GitLab

[GitLab](https://gitlab.com/) هو أداة دورة حياة DevOps قائمة على الويب توفر مدير مستودعات Git، وتقدم ميزات الويكي، وتتبع المشكلات، ومسار التكامل المستمر/التثبيت المستمر.

> **بيانات الاعتماد**
>
> يمكن العثور على معلومات المصادقة لهذه العقدة [هنا](/integrations/builtin/credentials/gitlab.md).

> **أمثلة وقوالب**
>
> للحصول على أمثلة الاستخدام والقوالب لمساعدتك على البدء، ارجع إلى صفحة [تكاملات GitLab المُحفِّزة](https://n8n.io/integrations/gitlab-trigger/) الخاصة بـ n8n.

## الأحداث

*   تعليق
*   مشكلات سرية
*   تعليقات سرية
*   عمليات النشر
*   مشكلة
*   مهمة
*   طلب دمج
*   مسار
*   دفع
*   إصدار
*   وسم
*   صفحة ويكي

## الموارد ذات الصلة

يوفر n8n عقدة تطبيق لـ GitLab. يمكنك العثور على وثائق العقدة [هنا](/integrations/builtin/app-nodes/n8n-nodes-base.gitlab.md).

عرض [أمثلة سير العمل والمحتوى ذي الصلة](https://n8n.io/integrations/gitlab-trigger/) على موقع n8n الإلكتروني.

ارجع إلى [وثائق GitLab](https://docs.gitlab.com/api/rest/) للحصول على تفاصيل حول واجهة برمجة التطبيقات الخاصة بهم.
