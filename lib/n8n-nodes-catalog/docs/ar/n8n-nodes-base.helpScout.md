# عقدة Help Scout

استخدم عقدة Help Scout (أداة مساعدة العملاء) لأتمتة العمل في Help Scout، ودمج Help Scout مع تطبيقات أخرى. تدعم n8n بشكل مدمج مجموعة واسعة من ميزات Help Scout، بما في ذلك إنشاء المحادثات والعملاء وتحديثهم وحذفهم واسترجاعهم.

تجد في هذه الصفحة قائمة بالعمليات التي تدعمها عقدة Help Scout وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد Help Scout](/integrations/builtin/credentials/helpscout.md) للحصول على إرشادات حول إعداد المصادقة.

## العمليات

*   المحادثة
    *   إنشاء محادثة جديدة
    *   حذف محادثة
    *   استرجاع محادثة
    *   استرجاع جميع المحادثات
*   العميل
    *   إنشاء عميل جديد
    *   استرجاع عميل
    *   استرجاع جميع العملاء
    *   استرجاع تعريفات خصائص العميل
    *   تحديث عميل
*   صندوق البريد
    *   استرجاع بيانات صندوق بريد
    *   استرجاع جميع صناديق البريد
*   الخيط
    *   إنشاء خيط دردشة جديد
    *   استرجاع جميع خيوط الدردشة

## ماذا تفعل إذا كانت عمليتك غير مدعومة

إذا كانت هذه العقدة لا تدعم العملية التي ترغب في تنفيذها، يمكنك استخدام [عقدة طلب HTTP (HTTP Request)](/integrations/builtin/core-nodes/n8n-nodes-base.httprequest/index.md) لاستدعاء واجهة برمجة تطبيقات الخدمة (API).

يمكنك استخدام بيانات الاعتماد التي أنشأتها لهذه الخدمة في عقدة طلب HTTP:

1.  في عقدة طلب HTTP، اختر **المصادقة** > **نوع بيانات الاعتماد المُحددة مسبقًا**.
2.  اختر الخدمة التي ترغب في الاتصال بها.
3.  اختر بيانات الاعتماد الخاصة بك.

ارجع إلى [عمليات API المخصصة](/integrations/custom-operations.md) لمزيد من المعلومات.
