# عقدة مُحفِّز Microsoft OneDrive

استخدم عقدة مُحفِّز Microsoft OneDrive للاستجابة للأحداث في [Microsoft OneDrive](https://www.microsoft.com/en-us/microsoft-365/onedrive/online-cloud-storage) ودمج Microsoft OneDrive مع تطبيقات أخرى. يدعم n8n أحداث الملفات والمجلدات في OneDrive بشكل مدمج.

في هذه الصفحة، ستجد قائمة بالأحداث التي يمكن لعقدة مُحفِّز Microsoft OneDrive الاستجابة لها وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> يمكنك العثور على معلومات المصادقة لهذه العقدة [هنا](/integrations/builtin/credentials/microsoft.md).

> **دعم السحابة الحكومية**
>
> إذا كنت تستخدم مستأجر سحابة حكومية (حكومة الولايات المتحدة، وزارة الدفاع الأمريكية، أو الصين)، فتأكد من تحديد **Microsoft Graph API Base URL** المناسب في تهيئة بيانات اعتماد Microsoft الخاصة بك.

> **أمثلة وقوالب**
>
> للحصول على أمثلة استخدام وقوالب لمساعدتك على البدء، ارجع إلى صفحة [تكاملات Microsoft OneDrive](https://n8n.io/integrations/microsoft-onedrive-trigger/) الخاصة بـ n8n.

## الأحداث

*   عند إنشاء ملف
*   عند تحديث ملف
*   عند إنشاء مجلد
*   عند تحديث مجلد

## الموارد ذات الصلة

يوفر n8n عقدة تطبيق لـ Microsoft OneDrive. يمكنك العثور على وثائق العقدة [هنا](/integrations/builtin/app-nodes/n8n-nodes-base.microsoftonedrive.md).

شاهد [أمثلة سير العمل والمحتوى ذي الصلة](https://n8n.io/integrations/microsoft-onedrive-trigger/) على موقع n8n الإلكتروني.

ارجع إلى [وثائق OneDrive API من Microsoft](https://learn.microsoft.com/en-us/onedrive/developer/rest-api/) لمزيد من المعلومات حول الخدمة.
