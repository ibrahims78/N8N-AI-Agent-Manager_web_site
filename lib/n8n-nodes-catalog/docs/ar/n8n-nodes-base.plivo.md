# عقدة Plivo

استخدم عقدة Plivo لأتمتة سير العمل في Plivo، ودمج Plivo مع تطبيقات أخرى. تدعم n8n بشكل مدمج مجموعة واسعة من ميزات Plivo، بما في ذلك إجراء المكالمات وإرسال رسائل SMS/MMS.

ستجد في هذه الصفحة قائمة بالعمليات التي تدعمها عقدة Plivo وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد Plivo](/integrations/builtin/credentials/plivo.md) للحصول على إرشادات حول إعداد المصادقة.

## العمليات

*   المكالمة
    *   إجراء مكالمة صوتية
*   رسائل MMS
    *   إرسال رسالة MMS (الولايات المتحدة/كندا فقط)
*   رسائل SMS
    *   إرسال رسالة SMS.
