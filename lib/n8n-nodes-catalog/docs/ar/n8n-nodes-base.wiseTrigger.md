# العقدة المُحفِّزة Wise

تسمح لك Wise (خدمة تحويل الأموال الدولية) بتحويل الأموال إلى الخارج بتكلفة منخفضة، واستلام الأموال باستخدام تفاصيل الحسابات الدولية، وتتبع المعاملات على هاتفك.

> **بيانات الاعتماد**
>
> يمكنك العثور على معلومات المصادقة لهذه العقدة [هنا](/integrations/builtin/credentials/wise.md).

> **أمثلة وقوالب**
>
> للحصول على أمثلة الاستخدام والقوالب لمساعدتك على البدء، ارجع إلى صفحة [تكاملات Wise Trigger](https://n8n.io/integrations/wise-trigger/) الخاصة بـ n8n.

## الأحداث

-   يتم تحفيزه في كل مرة يُضاف فيها رصيد إلى حساب
-   يتم تحفيزه في كل مرة يُضاف فيها رصيد إلى حساب أو يُخصم منه
-   يتم تحفيزه في كل مرة يتم فيها تحديث قائمة الحالات النشطة لعملية تحويل
-   يتم تحفيزه في كل مرة يتم فيها تحديث حالة عملية تحويل
