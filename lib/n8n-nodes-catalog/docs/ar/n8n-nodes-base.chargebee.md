# عقدة Chargebee

استخدم عقدة Chargebee لأتمتة سير العمل في Chargebee، ودمج Chargebee مع التطبيقات الأخرى. تدعم n8n بشكل مدمج مجموعة واسعة من ميزات Chargebee، بما في ذلك إنشاء العملاء، وإرجاع الفواتير، وإلغاء الاشتراكات.

في هذه الصفحة، ستجد قائمة بالعمليات التي تدعمها عقدة Chargebee وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد Chargebee](/integrations/builtin/credentials/chargebee.md) للحصول على إرشادات حول إعداد المصادقة.
