# عقدة GitHub المُحفِّزة

توفر [GitHub](https://github.com/) استضافة لتطوير البرمجيات والتحكم بالإصدارات باستخدام Git. تقدم المنصة وظائف التحكم الموزع بالإصدارات وإدارة شيفرة المصدر (SCM) الخاصة بـ Git، بالإضافة إلى التحكم بالوصول والعديد من ميزات التعاون مثل تتبع الأخطاء، طلبات الميزات، إدارة المهام، والويكيات لكل مشروع.

> **بيانات الاعتماد**
>
> يمكن العثور على معلومات المصادقة لهذه العقدة [هنا](/integrations/builtin/credentials/github.md).

> **أمثلة وقوالب**
>
> للحصول على أمثلة استخدام وقوالب لمساعدتك على البدء، يُرجى الرجوع إلى صفحة [تكاملات GitHub Trigger](https://n8n.io/integrations/github-trigger/) الخاصة بـ n8n.

## الموارد ذات الصلة

توفر n8n عقدة تطبيق لـ GitHub. يمكن العثور على توثيق العقدة [هنا](/integrations/builtin/app-nodes/n8n-nodes-base.github.md).

اطلع على [أمثلة سير العمل والمحتوى ذي الصلة](https://n8n.io/integrations/github-trigger/) على موقع n8n الإلكتروني.

ارجع إلى [توثيق GitHub](https://docs.github.com/en/rest) للحصول على تفاصيل حول واجهة برمجة التطبيقات (API) الخاصة بهم.
