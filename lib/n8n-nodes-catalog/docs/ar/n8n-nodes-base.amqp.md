# عقدة AMQP Sender

استخدم عقدة AMQP Sender لأتمتة سير العمل في AMQP Sender (بروتوكول طابور الرسائل المتقدم)، ودمج AMQP Sender مع تطبيقات أخرى. تدعم n8n بشكل مدمج مجموعة واسعة من ميزات AMQP Sender، بما في ذلك إرسال الرسائل.

ستجد في هذه الصفحة قائمة بالعمليات التي تدعمها عقدة AMQP Sender وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد AMQP Sender](/integrations/builtin/credentials/amqp.md) للحصول على إرشادات حول إعداد المصادقة.
