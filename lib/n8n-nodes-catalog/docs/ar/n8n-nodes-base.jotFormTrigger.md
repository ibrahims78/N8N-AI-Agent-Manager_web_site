# العقدة المُحفِّزة Jotform Trigger

Jotform (خدمة بناء النماذج عبر الإنترنت) هي خدمة لإنشاء النماذج عبر الإنترنت. ينشئ برنامج Jotform النماذج باستخدام أداة إنشاء بالسحب والإفلات وخيار لتشفير بيانات المستخدم.

> **بيانات الاعتماد**
>
> يمكنك العثور على معلومات المصادقة لهذه العقدة [هنا](/integrations/builtin/credentials/jotform.md).

> **أمثلة وقوالب**
>
> للاطلاع على أمثلة الاستخدام والقوالب لمساعدتك على البدء، ارجع إلى صفحة [تكاملات Jotform Trigger](https://n8n.io/integrations/jotform-trigger/) الخاصة بـ n8n.
