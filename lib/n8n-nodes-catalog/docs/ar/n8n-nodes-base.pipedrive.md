# عقدة Pipedrive

استخدم عقدة Pipedrive لأتمتة سير العمل في Pipedrive، ودمج Pipedrive مع تطبيقات أخرى. تدعم n8n بشكل مدمج مجموعة واسعة من ميزات Pipedrive، بما في ذلك إنشاء وتحديث وحذف والحصول على الأنشطة والملفات والملاحظات والمؤسسات والعملاء المحتملين.

ستجد في هذه الصفحة قائمة بالعمليات التي تدعمها عقدة Pipedrive وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد Pipedrive](/integrations/builtin/credentials/pipedrive.md) للحصول على إرشادات حول إعداد المصادقة.

> **يمكن استخدام هذه العقدة كأداة ذكاء اصطناعي**
>
> يمكن استخدام هذه العقدة لتعزيز قدرات وكيل الذكاء الاصطناعي. عند استخدامها بهذه الطريقة، يمكن تعيين العديد من المعاملات تلقائيًا، أو بمعلومات موجهة بواسطة الذكاء الاصطناعي - اكتشف المزيد في [وثائق معاملات أداة الذكاء الاصطناعي](/advanced-ai/examples/using_the_fromai-function.md).

## ماذا تفعل إذا كانت عمليتك غير مدعومة

إذا كانت هذه العقدة لا تدعم العملية التي ترغب في تنفيذها، يمكنك استخدام [عقدة HTTP Request](/integrations/builtin/core-nodes/n8n-nodes-base.httprequest/index.md) لاستدعاء واجهة برمجة تطبيقات الخدمة.

يمكنك استخدام بيانات الاعتماد التي أنشأتها لهذه الخدمة في عقدة HTTP Request:

1.  في عقدة HTTP Request، حدد **المصادقة** > **نوع بيانات الاعتماد المُحددة مسبقًا**.
2.  حدد الخدمة التي ترغب في الاتصال بها.
3.  حدد بيانات الاعتماد الخاصة بك.

ارجع إلى [عمليات API المخصصة](/integrations/custom-operations.md) لمزيد من المعلومات.
