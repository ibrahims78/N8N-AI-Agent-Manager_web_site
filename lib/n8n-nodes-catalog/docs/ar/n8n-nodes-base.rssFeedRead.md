# RSS Read

استخدم عقدة RSS Read (قراءة RSS) لقراءة البيانات من خلاصات RSS المنشورة على الإنترنت.

## معاملات العقدة

- **URL**: أدخل URL لمنشور RSS الذي ترغب في قراءته.

## خيارات العقدة

* **Ignore SSL Issues**: اختر ما إذا كان يجب على n8n تجاهل التحقق من SSL/TLS (عند التفعيل) أو عدم تجاهلها (عند التعطيل).
