# عقدة المُحفِّز SurveyMonkey

[SurveyMonkey](https://www.surveymonkey.com/) هي منصة استبيانات قائمة على السحابة كخدمة (SaaS) عبر الإنترنت، وتوفر أيضاً مجموعة من البرامج الخلفية المدفوعة.

> **بيانات الاعتماد**
>
> يمكن العثور على معلومات المصادقة لهذه العقدة [هنا](/integrations/builtin/credentials/surveymonkey.md).

> **أمثلة وقوالب**
>
> للاطلاع على أمثلة الاستخدام والقوالب لمساعدتك على البدء، ارجع إلى صفحة [تكاملات SurveyMonkey Trigger](https://n8n.io/integrations/surveymonkey-trigger/) الخاصة بـ n8n.
