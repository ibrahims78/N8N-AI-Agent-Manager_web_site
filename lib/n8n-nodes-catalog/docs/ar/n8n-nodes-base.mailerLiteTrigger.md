# العقدة المُحفِّزة MailerLite

[MailerLite](https://www.mailerlite.com/) هو حل للتسويق عبر البريد الإلكتروني يوفر لك محرر محتوى سهل الاستخدام، وإدارة مبسطة للمشتركين، وتقارير حملات تتضمن أهم الإحصائيات.

في هذه الصفحة، ستجد قائمة بالأحداث التي يمكن أن تستجيب لها العقدة المُحفِّزة MailerLite، وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> يمكنك العثور على معلومات المصادقة لهذه العقدة [هنا](/integrations/builtin/credentials/mailerlite.md).

> **أمثلة وقوالب**
>
> للحصول على أمثلة الاستخدام والقوالب لمساعدتك على البدء، ارجع إلى صفحة [تكاملات MailerLite Trigger](https://n8n.io/integrations/mailerlite-trigger/) الخاصة بـ n8n.

## الأحداث

- إرسال حملة
- إضافة مشترك إلى مجموعة
- اكتمال أتمتة المشترك
- تفعيل أتمتة المشترك
- ارتداد المشترك
- إنشاء مشترك
- شكوى المشترك
- إزالة مشترك من مجموعة
- إلغاء اشتراك المشترك
- تحديث المشترك
