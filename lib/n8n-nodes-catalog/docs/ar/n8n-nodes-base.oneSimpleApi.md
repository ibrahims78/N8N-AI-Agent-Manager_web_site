# عقدة One Simple API

استخدم عقدة One Simple API لأتمتة سير العمل في One Simple API، ودمج One Simple API مع تطبيقات أخرى. تدعم n8n مجموعة واسعة من ميزات One Simple API بشكل مدمج، بما في ذلك الحصول على الملفات الشخصية، واسترداد المعلومات، وتوليد الأدوات المساعدة.

ستجد في هذه الصفحة قائمة بالعمليات التي تدعمها عقدة One Simple API وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد One Simple API](/integrations/builtin/credentials/onesimpleapi.md) للحصول على إرشادات حول إعداد المصادقة.

## العمليات

*   المعلومات
    *   تحويل قيمة بين العملات
    *   استرداد بيانات تعريف الصورة من عنوان URL
*   الملف الشخصي الاجتماعي
    *   الحصول على تفاصيل حول ملف شخصي على Instagram
    *   الحصول على تفاصيل حول فنان على Spotify
*   الأداة المساعدة
    *   توسيع عنوان URL مختصر
    *   توليد رمز QR
    *   التحقق من صحة عنوان بريد إلكتروني
*   الموقع الإلكتروني
    *   توليد ملف PDF من صفحة ويب
    *   الحصول على معلومات تحسين محركات البحث (SEO) من موقع ويب
    *   إنشاء لقطة شاشة من صفحة ويب
