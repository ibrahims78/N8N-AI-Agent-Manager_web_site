# عقدة Stackby

استخدم عقدة Stackby لأتمتة سير العمل في Stackby، ودمج Stackby مع تطبيقات أخرى. تدعم n8n مجموعة واسعة من ميزات Stackby بشكل مدمج، بما في ذلك الإضافة والحذف والإدراج والقراءة.

ستجد في هذه الصفحة قائمة بالعمليات التي تدعمها عقدة Stackby وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد Stackby](/integrations/builtin/credentials/stackby.md) للحصول على إرشادات حول إعداد المصادقة.
