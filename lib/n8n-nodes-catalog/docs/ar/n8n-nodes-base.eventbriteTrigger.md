# عقدة مُحفِّز Eventbrite

[Eventbrite](https://www.eventbrite.com/) هو موقع ويب لإدارة الفعاليات وبيع التذاكر. تتيح الخدمة للمستخدمين تصفح الفعاليات المحلية وإنشائها والترويج لها.

> **بيانات الاعتماد**
>
> يمكنك العثور على معلومات المصادقة لهذه العقدة [هنا](/integrations/builtin/credentials/eventbrite.md).

> **أمثلة وقوالب**
>
> للاطلاع على أمثلة الاستخدام والقوالب لمساعدتك على البدء، ارجع إلى صفحة [تكاملات مُحفِّز Eventbrite](https://n8n.io/integrations/eventbrite-trigger/) الخاصة بـ n8n.
