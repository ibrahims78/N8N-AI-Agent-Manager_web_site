# العقدة المُحفِّزة Pipedrive

[Pipedrive](https://www.pipedrive.com/) هي شركة برمجيات مبيعات قائمة على السحابة (cloud-based sales software) تهدف إلى تحسين إنتاجية الشركات من خلال استخدام برمجياتها.

> **بيانات الاعتماد**
>
> يمكنك العثور على معلومات المصادقة لهذه العقدة [هنا](/integrations/builtin/credentials/pipedrive.md).

> **أمثلة وقوالب**
>
> لأمثلة الاستخدام والقوالب لمساعدتك على البدء، ارجع إلى صفحة [تكاملات Pipedrive Trigger](https://n8n.io/integrations/pipedrive-trigger/) الخاصة بـ n8n.
