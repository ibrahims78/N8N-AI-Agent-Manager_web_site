# TOTP

توفر عقدة TOTP طريقة لتوليد TOTP (كلمة مرور لمرة واحدة تعتمد على الوقت).

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد TOTP](/integrations/builtin/credentials/totp.md) للحصول على إرشادات حول إعداد المصادقة.

## معاملات العقدة

> **يمكن استخدام هذه العقدة كأداة للذكاء الاصطناعي**
>
> يمكن استخدام هذه العقدة لتعزيز قدرات وكيل الذكاء الاصطناعي. عند استخدامها بهذه الطريقة، يمكن تعيين العديد من المعاملات تلقائياً، أو بمعلومات موجهة بواسطة الذكاء الاصطناعي - اكتشف المزيد في [توثيق معاملات أداة الذكاء الاصطناعي](/advanced-ai/examples/using-the-fromai-function.md).

قم بتكوين هذه العقدة باستخدام هذه المعاملات.

### بيانات الاعتماد للاتصال بـ

حدد أو أنشئ [بيانات اعتماد TOTP](/integrations/builtin/credentials/totp.md) لتستخدمها العقدة.

### العملية

**Generate Secret** هي العملية الوحيدة المدعومة حالياً.

## خيارات العقدة

استخدم هذه **الخيارات** لتكوين العقدة بشكل إضافي.

### الخوارزمية

حدد خوارزمية تجزئة HMAC المراد استخدامها. الافتراضي هو SHA1.

### الأرقام

أدخل عدد الأرقام في الرمز المُولَّد. الافتراضي هو `6`.

### المدة

أدخل عدد الثواني التي يكون فيها TOTP صالحاً لـ. الافتراضي هو `30`.
