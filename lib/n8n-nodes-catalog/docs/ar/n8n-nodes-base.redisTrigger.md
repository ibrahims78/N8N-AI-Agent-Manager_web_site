# عقدة Redis Trigger

[Redis](https://redis.io/) هو مخزن هياكل بيانات مفتوح المصدر في الذاكرة، يُستخدم كقاعدة بيانات، وذاكرة تخزين مؤقت (cache)، ووسيط رسائل (message broker).

استخدم عقدة Redis Trigger للاشتراك في قناة Redis. يبدأ سير العمل (workflow) كلما استقبلت القناة رسالة جديدة.

> **بيانات الاعتماد**
>
> يمكنك العثور على معلومات المصادقة (authentication) لهذه العقدة [هنا](/integrations/builtin/credentials/redis.md).

> **أمثلة وقوالب**
>
> للاطلاع على أمثلة الاستخدام والقوالب لمساعدتك على البدء، ارجع إلى صفحة [تكاملات Redis Trigger](https://n8n.io/integrations/redis-trigger/) الخاصة بـ n8n.
