# عقدة Wufoo المُحفِّز

[Wufoo](https://wufoo.com) هو منشئ نماذج عبر الإنترنت يساعدك على إنشاء نماذج HTML مخصصة دون كتابة أي كود.

> **بيانات الاعتماد**
>
> يمكنك العثور على معلومات المصادقة لهذه العقدة [هنا](/integrations/builtin/credentials/wufoo.md).

> **أمثلة وقوالب**
>
> للحصول على أمثلة استخدام وقوالب لمساعدتك على البدء، ارجع إلى صفحة [تكاملات Wufoo Trigger](https://n8n.io/integrations/wufoo-trigger/) الخاصة بـ n8n.
