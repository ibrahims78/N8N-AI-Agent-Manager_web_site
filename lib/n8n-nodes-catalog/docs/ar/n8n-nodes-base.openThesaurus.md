# عقدة OpenThesaurus

استخدم عقدة OpenThesaurus لأتمتة سير العمل في OpenThesaurus، ودمج OpenThesaurus مع تطبيقات أخرى. تدعم n8n البحث عن المرادفات للكلمات الألمانية.

في هذه الصفحة، ستجد قائمة بالعمليات التي تدعمها عقدة OpenThesaurus وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> لا تتطلب عقدة OpenThesaurus المصادقة.

## العمليات

*   الحصول على مرادفات لكلمة ألمانية باللغة الألمانية
