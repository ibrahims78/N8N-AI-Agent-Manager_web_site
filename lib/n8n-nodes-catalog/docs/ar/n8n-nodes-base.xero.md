# عقدة Xero

استخدم عقدة Xero لأتمتة سير العمل في Xero، ودمج Xero مع تطبيقات أخرى. تدعم n8n بشكل مدمج مجموعة واسعة من ميزات Xero، بما في ذلك إنشاء جهات الاتصال والفواتير وتحديثها والحصول عليها.

ستجد في هذه الصفحة قائمة بالعمليات التي تدعمها عقدة Xero وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد Xero](/integrations/builtin/credentials/xero.md) للحصول على إرشادات حول إعداد المصادقة.

## العمليات

*   جهة الاتصال
    *   إنشاء جهة اتصال
    *   الحصول على جهة اتصال
    *   الحصول على جميع جهات الاتصال
    *   تحديث جهة اتصال
*   الفاتورة
    *   إنشاء فاتورة
    *   الحصول على فاتورة
    *   الحصول على جميع الفواتير
    *   تحديث فاتورة

## الموارد ذات الصلة

ارجع إلى [توثيق API الخاص بـ Xero](https://developer.xero.com/documentation/api/accounting/overview) لمزيد من المعلومات حول الخدمة.

## ماذا تفعل إذا كانت عمليتك غير مدعومة

إذا كانت هذه العقدة لا تدعم العملية التي ترغب في تنفيذها، يمكنك استخدام [عقدة HTTP Request](/integrations/builtin/core-nodes/n8n-nodes-base.httprequest/index.md) لاستدعاء API الخاص بالخدمة.

يمكنك استخدام بيانات الاعتماد التي أنشأتها لهذه الخدمة في عقدة HTTP Request:

1.  في عقدة HTTP Request، حدد **المصادقة** > **نوع بيانات الاعتماد المُحددة مسبقًا**.
2.  حدد الخدمة التي ترغب في الاتصال بها.
3.  حدد بيانات الاعتماد الخاصة بك.

ارجع إلى [عمليات API المخصصة](/integrations/custom-operations.md) لمزيد من المعلومات.
