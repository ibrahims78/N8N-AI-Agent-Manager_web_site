# عقدة Odoo

استخدم عقدة Odoo لأتمتة سير العمل في Odoo، ودمج Odoo مع التطبيقات الأخرى. تدعم n8n بشكل مدمج مجموعة واسعة من ميزات Odoo، بما في ذلك إنشاء، تحديث، حذف، واسترداد العقود، الموارد، والفرص.

في هذه الصفحة، ستجد قائمة بالعمليات التي تدعمها عقدة Odoo وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد Odoo](/integrations/builtin/credentials/odoo.md) للحصول على إرشادات حول إعداد المصادقة.

> **يمكن استخدام هذه العقدة كأداة ذكاء اصطناعي**
>
> يمكن استخدام هذه العقدة لتعزيز قدرات وكيل الذكاء الاصطناعي. عند استخدامها بهذه الطريقة، يمكن تعيين العديد من المعاملات تلقائيًا، أو بمعلومات موجهة بواسطة الذكاء الاصطناعي - اكتشف المزيد في [توثيق معاملات أداة الذكاء الاصطناعي](/advanced-ai/examples/using-the-fromai-function.md).
