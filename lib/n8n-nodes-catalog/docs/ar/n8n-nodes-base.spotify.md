# عقدة Spotify

استخدم عقدة Spotify لأتمتة سير العمل في Spotify، ودمج Spotify مع تطبيقات أخرى. تدعم n8n مجموعة واسعة من ميزات Spotify بشكل مدمج، بما في ذلك الحصول على معلومات الألبومات والفنانين.

ستجد في هذه الصفحة قائمة بالعمليات التي تدعمها عقدة Spotify وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد Spotify](/integrations/builtin/credentials/spotify.md) للحصول على إرشادات حول إعداد المصادقة.

## العمليات

*   الألبوم
    *   الحصول على ألبوم بواسطة URI أو ID.
    *   الحصول على قائمة بإصدارات الألبومات الجديدة.
    *   الحصول على مسارات (tracks) ألبوم بواسطة URI أو ID.
    *   البحث عن ألبومات بواسطة كلمة مفتاحية.
*   الفنان
    *   الحصول على فنان بواسطة URI أو ID.
    *   الحصول على ألبومات فنان بواسطة URI أو ID.
    *   الحصول على فنانين مرتبطين بفنان بواسطة URI أو ID.
    *   الحصول على أفضل مسارات (tracks) فنان بواسطة URI أو ID.
    *   البحث عن فنانين بواسطة كلمة مفتاحية.
*   المكتبة
    *   الحصول على المسارات (tracks) التي أعجب بها المستخدم.
*   بياناتي
    *   الحصول على الفنانين الذين تتابعهم.
*   المشغل
    *   إضافة أغنية إلى قائمة الانتظار الخاصة بك.
    *   الحصول على المسار (track) الذي يتم تشغيله حاليًا.
    *   الانتقال إلى المسار (track) التالي.
    *   إيقاف الموسيقى مؤقتًا.
    *   الانتقال إلى الأغنية السابقة.
    *   الحصول على المسارات (tracks) التي تم تشغيلها مؤخرًا.
    *   استئناف التشغيل على الجهاز النشط الحالي.
    *   ضبط مستوى الصوت على الجهاز النشط الحالي.
    *   بدء تشغيل قائمة تشغيل (playlist) أو فنان أو ألبوم.
*   قائمة التشغيل (Playlist)
    *   إضافة مسارات (tracks) إلى قائمة تشغيل (playlist) بواسطة URI أو ID للمسار وقائمة التشغيل.
    *   إنشاء قائمة تشغيل (playlist) جديدة.
    *   الحصول على قائمة تشغيل (playlist) بواسطة URI أو ID.
    *   الحصول على مسارات (tracks) قائمة تشغيل (playlist) بواسطة URI أو ID.
    *   الحصول على قوائم تشغيل (playlists) المستخدم.
    *   إزالة مسارات (tracks) من قائمة تشغيل (playlist) بواسطة URI أو ID للمسار وقائمة التشغيل.
    *   البحث عن قوائم تشغيل (playlists) بواسطة كلمة مفتاحية.
*   المسار (Track)
    *   الحصول على مسار (track) بواسطة URI أو ID الخاص به.
    *   الحصول على ميزات الصوت لمسار (track) بواسطة URI أو ID.
    *   البحث عن مسارات (tracks) بواسطة كلمة مفتاحية.

## ماذا تفعل إذا كانت عمليتك غير مدعومة

إذا كانت هذه العقدة لا تدعم العملية التي ترغب في تنفيذها، يمكنك استخدام [عقدة طلب HTTP (HTTP Request)](/integrations/builtin/core-nodes/n8n-nodes-base.httprequest/index.md) لاستدعاء واجهة برمجة تطبيقات (API) الخدمة.

يمكنك استخدام بيانات الاعتماد التي أنشأتها لهذه الخدمة في عقدة طلب HTTP (HTTP Request):

1.  في عقدة طلب HTTP (HTTP Request)، حدد **المصادقة** > **نوع بيانات الاعتماد المُحددة مسبقًا**.
2.  حدد الخدمة التي ترغب في الاتصال بها.
3.  حدد بيانات الاعتماد الخاصة بك.

ارجع إلى [عمليات API المخصصة](/integrations/custom-operations.md) لمزيد من المعلومات.
