# عقدة Storyblok

استخدم عقدة Storyblok لأتمتة سير العمل في Storyblok، ودمج Storyblok مع تطبيقات أخرى. تدعم n8n بشكل مدمج مجموعة واسعة من ميزات Storyblok، بما في ذلك جلب القصص وحذفها ونشرها.

في هذه الصفحة، ستجد قائمة بالعمليات التي تدعمها عقدة Storyblok وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد Storyblok](/integrations/builtin/credentials/storyblok.md) للحصول على إرشادات حول إعداد المصادقة.

## العمليات

### واجهة برمجة تطبيقات المحتوى (Content API)

- **القصة**
    - جلب قصة
    - جلب جميع القصص

### واجهة برمجة تطبيقات الإدارة (Management API)

- **القصة**
    - حذف قصة
    - جلب قصة
    - جلب جميع القصص
    - نشر قصة
    - إلغاء نشر قصة
