# عقدة CircleCI

استخدم عقدة CircleCI (منصة التكامل المستمر والتسليم المستمر) لأتمتة سير العمل في CircleCI، ودمج CircleCI مع تطبيقات أخرى. تدعم n8n بشكل مدمج مجموعة واسعة من ميزات CircleCI، بما في ذلك جلب سير العمل وتشغيله.

ستجد في هذه الصفحة قائمة بالعمليات التي تدعمها عقدة CircleCI وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد CircleCI](/integrations/builtin/credentials/circleci.md) للحصول على إرشادات حول إعداد المصادقة.
