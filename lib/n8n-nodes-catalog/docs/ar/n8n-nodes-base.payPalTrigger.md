# العقدة المُحفِّزة PayPal

[PayPal](https://paypal.com) (خدمة دفع رقمية) هي خدمة دفع رقمية تدعم تحويل الأموال عبر الإنترنت يمكن للعملاء استخدامها عند التسوق عبر الإنترنت.

> **بيانات الاعتماد**
>
> يمكن العثور على معلومات المصادقة لهذه العقدة [هنا](/integrations/builtin/credentials/paypal.md).

> **أمثلة وقوالب**
>
> للاطلاع على أمثلة الاستخدام والقوالب لمساعدتك على البدء، ارجع إلى صفحة [تكاملات PayPal Trigger](https://n8n.io/integrations/paypal-trigger/) الخاصة بـ n8n.
