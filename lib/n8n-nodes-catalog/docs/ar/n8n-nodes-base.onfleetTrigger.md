# عقدة مُحفِّز Onfleet

[Onfleet](https://onfleet.com/) (منصة لوجستية) هي منصة لوجستية تقدم حلول توصيل الميل الأخير.

> **بيانات الاعتماد**
>
> يمكنك العثور على معلومات المصادقة لهذه العقدة [هنا](/integrations/builtin/credentials/onfleet.md).

> **أمثلة وقوالب**
>
> للاطلاع على أمثلة الاستخدام والقوالب لمساعدتك على البدء، ارجع إلى صفحة [تكاملات Onfleet Trigger](https://n8n.io/integrations/onfleet-trigger/) الخاصة بـ n8n.

## الأحداث

يُحفِّز سير عمل عند:

*   إلغاء اشتراك مستلم الرسائل النصية القصيرة
*   فقدان استجابة مستلم الرسائل النصية القصيرة
*   وصول المهمة
*   تعيين المهمة
*   استنساخ المهمة
*   اكتمال المهمة
*   إنشاء المهمة
*   تأخير المهمة
*   الوقت المقدر لوصول المهمة (ETA)
*   فشل المهمة
*   بدء المهمة
*   إلغاء تعيين المهمة
*   تحديث المهمة
*   إنشاء عامل
*   حذف عامل
*   واجب العامل
