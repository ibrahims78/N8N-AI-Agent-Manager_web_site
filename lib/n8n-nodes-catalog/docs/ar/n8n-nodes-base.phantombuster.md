# عقدة PhantomBuster

استخدم عقدة PhantomBuster لأتمتة سير العمل في PhantomBuster، ودمج PhantomBuster مع تطبيقات أخرى. تدعم n8n مجموعة واسعة من ميزات PhantomBuster بشكل مدمج، بما في ذلك إضافة الوكلاء وحذفهم والحصول عليهم.

ستجد في هذه الصفحة قائمة بالعمليات التي تدعمها عقدة PhantomBuster وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد PhantomBuster](/integrations/builtin/credentials/phantombuster.md) للحصول على إرشادات حول إعداد المصادقة.

## العمليات

*   الوكيل
    *   حذف وكيل بواسطة المعرّف (ID).
    *   الحصول على وكيل بواسطة المعرّف (ID).
    *   الحصول على جميع وكلاء مؤسسة المستخدم الحالي.
    *   الحصول على مخرجات أحدث حاوية لوكيل.
    *   إضافة وكيل إلى قائمة الانتظار للتشغيل.

## ماذا تفعل إذا كانت عمليتك غير مدعومة

إذا كانت هذه العقدة لا تدعم العملية التي ترغب في تنفيذها، يمكنك استخدام [عقدة HTTP Request](/integrations/builtin/core-nodes/n8n-nodes-base.httprequest/index.md) لاستدعاء واجهة برمجة تطبيقات (API) الخدمة.

يمكنك استخدام بيانات الاعتماد التي أنشأتها لهذه الخدمة في عقدة HTTP Request:

1.  في عقدة HTTP Request، حدد **المصادقة** > **نوع بيانات الاعتماد المُحددة مسبقًا**.
2.  حدد الخدمة التي ترغب في الاتصال بها.
3.  حدد بيانات الاعتماد الخاصة بك.

ارجع إلى [عمليات API المخصصة](/integrations/custom-operations.md) لمزيد من المعلومات.
