# عقدة مُحفِّز (Trigger) ConvertKit

[ConvertKit](https://www.convertkit.com/) هي منصة تسويق عبر البريد الإلكتروني متكاملة الميزات. استخدم ConvertKit لبناء قائمة بريد إلكتروني، وإرسال رسائل بريد إلكتروني جماعية، وأتمتة التسلسلات، وإنشاء شرائح، وبناء صفحات هبوط.

> **بيانات الاعتماد (Credentials)**
>
> يمكنك العثور على معلومات المصادقة (Authentication) لهذه العقدة (Node) [هنا](/integrations/builtin/credentials/convertkit.md).

> **أمثلة وقوالب (Templates)**
>
> للحصول على أمثلة الاستخدام والقوالب (Templates) لمساعدتك على البدء، ارجع إلى صفحة [تكاملات (Integrations) مُحفِّز (Trigger) ConvertKit](https://n8n.io/integrations/convertkit-trigger/) الخاصة بـ n8n.

## الأحداث (Events)

*   الاشتراك في نموذج (Form subscribe)
*   النقر على رابط (Link click)
*   شراء منتج (Product purchase)
*   إنشاء عملية شراء (Purchase created)
*   اكتمال عملية الشراء (Purchase complete)
*   اكتمال التسلسل (Sequence complete)
*   الاشتراك في تسلسل (Sequence subscribe)
*   تفعيل المشترك (Subscriber activated)
*   إلغاء اشتراك المشترك (Subscriber unsubscribe)
*   إضافة وسم (Tag add)
*   إزالة وسم (Tag Remove)
