# عقدة Zammad

استخدم عقدة Zammad (نظام إدارة التذاكر) لأتمتة سير العمل في Zammad، ودمج Zammad مع تطبيقات أخرى. تدعم n8n بشكل مدمج مجموعة واسعة من ميزات Zammad، بما في ذلك إنشاء المجموعات والمؤسسات واستردادها وحذفها.

في هذه الصفحة، ستجد قائمة بالعمليات التي تدعمها عقدة Zammad وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد Zammad](/integrations/builtin/credentials/zammad.md) للحصول على إرشادات حول إعداد المصادقة.
