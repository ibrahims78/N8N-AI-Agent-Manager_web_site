# عقدة QuestDB

استخدم عقدة QuestDB لأتمتة سير العمل في QuestDB، ودمج QuestDB مع تطبيقات أخرى. تدعم n8n تنفيذ استعلام SQL وإدراج صفوف في قاعدة بيانات باستخدام QuestDB.

في هذه الصفحة، ستجد قائمة بالعمليات التي تدعمها عقدة QuestDB وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد QuestDB](/integrations/builtin/credentials/questdb.md) للحصول على إرشادات حول إعداد المصادقة.

## العمليات

*   تنفذ استعلام SQL.
*   تُدرج صفوفًا في قاعدة البيانات.

## مرجع العقدة

### تحديد نوع بيانات العمود

لتحديد نوع بيانات العمود، ألحق اسم العمود بـ `:type`، حيث `type` هو نوع البيانات الذي تريده للعمود. على سبيل المثال، إذا كنت ترغب في تحديد النوع `int` للعمود **id** والنوع `text` للعمود **name**، يمكنك استخدام المقتطف التالي في حقل **Columns**: `id:int,name:text`.
