# عقدة HaloPSA

استخدم عقدة HaloPSA لأتمتة سير العمل في HaloPSA، ودمج HaloPSA مع تطبيقات أخرى. تدعم n8n مجموعة واسعة من ميزات HaloPSA بشكل مدمج، بما في ذلك إنشاء العملاء والمواقع والتذاكر وتحديثها وحذفها والحصول عليها.

في هذه الصفحة، ستجد قائمة بالعمليات التي تدعمها عقدة HaloPSA وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد HaloPSA](/integrations/builtin/credentials/halopsa.md) للحصول على إرشادات حول إعداد المصادقة.
