# عقدة Taiga

استخدم عقدة Taiga لأتمتة سير العمل في Taiga، ودمج Taiga مع تطبيقات أخرى. تدعم n8n بشكل مدمج مجموعة واسعة من ميزات Taiga، بما في ذلك إنشاء المشكلات وتحديثها وحذفها واستردادها.

ستجد في هذه الصفحة قائمة بالعمليات التي تدعمها عقدة Taiga وروابط لموارد إضافية.
