# العقدة المُحفِّزة Toggl

[Toggl](https://toggl.com/) هو تطبيق لتتبع الوقت يقدم خدمات تتبع الوقت وإعداد التقارير عبر الإنترنت من خلال موقعه الإلكتروني، بالإضافة إلى تطبيقات الجوال وسطح المكتب.

> **بيانات الاعتماد**
>
> يمكنك العثور على معلومات المصادقة لهذه العقدة [هنا](/integrations/builtin/credentials/toggl.md).

> **أمثلة وقوالب**
>
> للاطلاع على أمثلة الاستخدام والقوالب لمساعدتك على البدء، ارجع إلى صفحة [تكاملات Toggl Trigger](https://n8n.io/integrations/toggl-trigger/) الخاصة بـ n8n.
