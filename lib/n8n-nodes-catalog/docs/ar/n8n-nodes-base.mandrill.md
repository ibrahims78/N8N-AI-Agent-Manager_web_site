# عقدة Mandrill

استخدم عقدة Mandrill لأتمتة سير العمل في Mandrill، ودمج Mandrill مع تطبيقات أخرى. تدعم n8n إرسال الرسائل بناءً على القوالب أو HTML باستخدام Mandrill.

ستجد في هذه الصفحة قائمة بالعمليات التي تدعمها عقدة Mandrill وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد Mandrill](/integrations/builtin/credentials/mandrill.md) للحصول على إرشادات حول إعداد المصادقة.

## العمليات

*   الرسالة
    *   إرسال رسالة بناءً على قالب.
    *   إرسال رسالة بناءً على HTML.
