# عقدة PayPal

استخدم عقدة PayPal (باي بال) لأتمتة سير العمل في PayPal، ودمج PayPal مع تطبيقات أخرى. تدعم n8n مجموعة واسعة من ميزات PayPal بشكل مدمج، بما في ذلك إنشاء دفعة مدفوعات جماعية وإلغاء عناصر المدفوعات غير المطالب بها.

ستجد في هذه الصفحة قائمة بالعمليات التي تدعمها عقدة PayPal وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد PayPal](/integrations/builtin/credentials/paypal.md) للحصول على إرشادات حول إعداد المصادقة.

## العمليات

* المدفوعات
    * إنشاء دفعة مدفوعات جماعية
    * عرض تفاصيل دفعة المدفوعات الجماعية
* عنصر المدفوعات
    * إلغاء عنصر مدفوعات غير مطالب به
    * عرض تفاصيل عنصر المدفوعات
