# العقدة المُحفِّزة Box (بوكس)

[Box](https://www.box.com/) هي شركة حوسبة سحابية توفر مشاركة الملفات والتعاون وأدوات أخرى للعمل مع الملفات المرفوعة إلى خوادمها.

> **بيانات الاعتماد**
>
> يمكنك العثور على معلومات المصادقة لهذه العقدة [هنا](/integrations/builtin/credentials/box.md).

> **أمثلة وقوالب**
>
> للحصول على أمثلة استخدام وقوالب لمساعدتك على البدء، ارجع إلى صفحة [تكاملات Box Trigger](https://n8n.io/integrations/box-trigger/) الخاصة بـ n8n.

## ابحث عن Box Target ID الخاص بك

للحصول على Target ID الخاص بك في Box:

1.  افتح الملف/المجلد الذي ترغب في مراقبته.
2.  انسخ سلسلة الأحرف بعد `folder/` في عنوان URL الخاص بك. هذا هو الـ Target ID. على سبيل المثال، إذا كان عنوان URL هو `https://app.box.com/folder/12345`، فإن `12345` هو الـ Target ID.
3.  الصقه في حقل **Target ID** في n8n.
