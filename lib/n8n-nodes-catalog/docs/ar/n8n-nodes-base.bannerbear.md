# عقدة Bannerbear

استخدم عقدة Bannerbear لأتمتة سير العمل في Bannerbear، ودمج Bannerbear مع تطبيقات أخرى. تدعم n8n بشكل مدمج مجموعة واسعة من ميزات Bannerbear، بما في ذلك إنشاء الصور والقوالب واستردادها.

ستجد في هذه الصفحة قائمة بالعمليات التي تدعمها عقدة Bannerbear وروابط لموارد إضافية.

> ارجع إلى [بيانات اعتماد Bannerbear](/integrations/builtin/credentials/bannerbear.md) للحصول على إرشادات حول إعداد المصادقة.
