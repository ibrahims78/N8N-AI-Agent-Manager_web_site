# عقدة Redis Chat Memory

استخدم عقدة Redis Chat Memory (ذاكرة الدردشة في Redis) لاستخدام Redis كخادم [ذاكرة](/glossary.md#ai-memory).

في هذه الصفحة، ستجد قائمة بالعمليات التي تدعمها عقدة Redis Chat Memory، وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> يمكنك العثور على معلومات المصادقة لهذه العقدة [هنا](/integrations/builtin/credentials/redis.md).

> **معالجة المعاملات في العقد الفرعية**
>
> تتصرف العقد الفرعية بشكل مختلف عن العقد الأخرى عند معالجة عناصر متعددة باستخدام تعبير.
>
> تأخذ معظم العقد، بما في ذلك العقد الجذرية، أي عدد من العناصر كمدخلات، وتعالج هذه العناصر، وتُخرج النتائج. يمكنك استخدام التعبيرات للإشارة إلى عناصر المدخلات، وتقوم العقدة بحل التعبير لكل عنصر على حدة. على سبيل المثال، بالنظر إلى مدخلات تتكون من خمس قيم `name`، يتم حل التعبير `` لكل اسم على حدة.
>
> في العقد الفرعية، يتم حل التعبير دائمًا إلى العنصر الأول. على سبيل المثال، بالنظر إلى مدخلات تتكون من خمس قيم `name`، يتم حل التعبير `` دائمًا إلى الاسم الأول.

## معاملات العقدة

*   **Session Key**: أدخل المفتاح المراد استخدامه لتخزين الذاكرة في بيانات سير العمل.
*   **Session Time To Live**: استخدم هذا المعامل لجعل الجلسة تنتهي بعد عدد معين من الثواني.
*   **Context Window Length**: أدخل عدد التفاعلات السابقة التي يجب أخذها في الاعتبار للسياق.
