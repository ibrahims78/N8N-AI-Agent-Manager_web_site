# عقدة Microsoft Graph Security

استخدم عقدة Microsoft Graph Security لأتمتة سير العمل في Microsoft Graph Security، وتكامل Microsoft Graph Security مع التطبيقات الأخرى. تدعم n8n مجموعة واسعة من ميزات Microsoft Graph Security بشكل مدمج، بما في ذلك الحصول على النقاط والملفات الشخصية وتحديثها.

في هذه الصفحة، ستجد قائمة بالعمليات التي تدعمها عقدة Microsoft Graph Security وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد Microsoft](/integrations/builtin/credentials/microsoft.md) للحصول على إرشادات حول إعداد المصادقة.

> **دعم السحابة الحكومية**
>
> إذا كنت تستخدم مستأجر سحابة حكومية (حكومة الولايات المتحدة، وزارة الدفاع الأمريكية، أو الصين)، فتأكد من تحديد **عنوان URL الأساسي لواجهة برمجة تطبيقات Microsoft Graph** المناسب في إعدادات بيانات اعتماد Microsoft الخاصة بك.

## ماذا تفعل إذا كانت عمليتك غير مدعومة

إذا كانت هذه العقدة لا تدعم العملية التي ترغب في تنفيذها، يمكنك استخدام [عقدة طلب HTTP](/integrations/builtin/core-nodes/n8n-nodes-base.httprequest/index.md) لاستدعاء واجهة برمجة تطبيقات الخدمة.

يمكنك استخدام بيانات الاعتماد التي أنشأتها لهذه الخدمة في عقدة طلب HTTP:

1.  في عقدة طلب HTTP، حدد **المصادقة** > **نوع بيانات الاعتماد المُحددة مسبقًا**.
2.  حدد الخدمة التي ترغب في الاتصال بها.
3.  حدد بيانات الاعتماد الخاصة بك.

ارجع إلى [عمليات API المخصصة](/integrations/custom-operations.md) لمزيد من المعلومات.
