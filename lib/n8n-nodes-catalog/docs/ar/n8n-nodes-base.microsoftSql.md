# عقدة Microsoft SQL

استخدم عقدة Microsoft SQL (مايكروسوفت إس كيو إل) لأتمتة سير العمل في Microsoft SQL، وتكامل Microsoft SQL مع التطبيقات الأخرى. تدعم n8n بشكل مدمج مجموعة واسعة من ميزات Microsoft SQL، بما في ذلك تنفيذ استعلامات SQL، وإدراج صفوف في قاعدة البيانات.

في هذه الصفحة، ستجد قائمة بالعمليات التي تدعمها عقدة Microsoft SQL وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد Microsoft SQL](/integrations/builtin/credentials/microsoftsql.md) للحصول على إرشادات حول إعداد المصادقة.

> **يمكن استخدام هذه العقدة كأداة ذكاء اصطناعي**
>
> يمكن استخدام هذه العقدة لتعزيز قدرات وكيل الذكاء الاصطناعي. عند استخدامها بهذه الطريقة، يمكن تعيين العديد من المعاملات تلقائيًا، أو بمعلومات موجهة بواسطة الذكاء الاصطناعي - اكتشف المزيد في [توثيق معاملات أداة الذكاء الاصطناعي](/advanced-ai/examples/using-the-fromai-function.md).
