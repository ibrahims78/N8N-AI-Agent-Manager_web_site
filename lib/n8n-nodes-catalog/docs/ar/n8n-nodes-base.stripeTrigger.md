# عقدة Stripe المُحفِّزة (Trigger)

Stripe هي مجموعة من واجهات برمجة التطبيقات (APIs) للمدفوعات التي تدعم التجارة للأعمال التجارية عبر الإنترنت.

> **بيانات الاعتماد**
>
> يمكنك العثور على معلومات المصادقة لهذه العقدة [هنا](/integrations/builtin/credentials/stripe.md).

> **أمثلة وقوالب**
>
> للاطلاع على أمثلة الاستخدام والقوالب لمساعدتك على البدء، ارجع إلى صفحة [تكاملات Stripe Trigger](https://n8n.io/integrations/stripe-trigger/) الخاصة بـ n8n.
