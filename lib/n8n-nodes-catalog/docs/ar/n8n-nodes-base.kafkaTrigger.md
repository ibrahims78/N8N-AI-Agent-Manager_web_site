# عقدة Kafka (كافكا) المُحفِّزة

[Kafka](https://kafka.apache.org/) هي منصة بث أحداث موزعة مفتوحة المصدر يمكن استخدامها لمسارات البيانات عالية الأداء، وتحليلات البث، وتكامل البيانات، والتطبيقات بالغة الأهمية.

> **بيانات الاعتماد**
>
> يمكن العثور على معلومات المصادقة لهذه العقدة [هنا](/integrations/builtin/credentials/kafka.md).

> **أمثلة وقوالب**
>
> للاطلاع على أمثلة الاستخدام والقوالب لمساعدتك على البدء، ارجع إلى صفحة [تكاملات Kafka Trigger](https://n8n.io/integrations/kafka-trigger/) الخاصة بـ n8n.
