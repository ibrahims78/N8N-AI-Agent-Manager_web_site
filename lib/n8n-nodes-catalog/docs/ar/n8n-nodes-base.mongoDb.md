# عقدة MongoDB

استخدم عقدة MongoDB لأتمتة سير العمل في MongoDB، ودمج MongoDB مع تطبيقات أخرى. تدعم n8n مجموعة واسعة من ميزات MongoDB بشكل مدمج، بما في ذلك تجميع المستندات، وتحديثها، والبحث عنها، وحذفها، واستردادها، بالإضافة إلى إنشاء فهارس البحث وتحديثها وسردها وإسقاطها. تستخدم جميع العمليات في هذه العقدة [برنامج تشغيل MongoDB Node](https://www.mongodb.com/docs/drivers/node/current/).

ستجد في هذه الصفحة قائمة بالعمليات التي تدعمها عقدة MongoDB وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد MongoDB](/integrations/builtin/credentials/mongodb.md) للحصول على إرشادات حول إعداد المصادقة.

> **يمكن استخدام هذه العقدة كأداة ذكاء اصطناعي**
>
> يمكن استخدام هذه العقدة لتعزيز قدرات وكيل الذكاء الاصطناعي. عند استخدامها بهذه الطريقة، يمكن تعيين العديد من المعاملات تلقائيًا، أو بمعلومات موجهة بواسطة الذكاء الاصطناعي - اكتشف المزيد في [توثيق معاملات أداة الذكاء الاصطناعي](/advanced-ai/examples/using-the-fromai-function.md).
