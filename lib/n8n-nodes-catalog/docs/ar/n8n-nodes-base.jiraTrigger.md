# عقدة مُحفِّز Jira

[Jira](https://www.atlassian.com/software/jira) (جيرا) هو منتج خاص لتتبع المشكلات طورته Atlassian، ويسمح بتتبع الأخطاء وإدارة المشاريع الرشيقة.

> **بيانات الاعتماد**
>
> يمكن العثور على معلومات المصادقة لهذه العقدة [هنا](/integrations/builtin/credentials/jira.md).

> **أمثلة وقوالب**
>
> للاطلاع على أمثلة الاستخدام والقوالب لمساعدتك على البدء، ارجع إلى صفحة [تكاملات مُحفِّز Jira](https://n8n.io/integrations/jira-trigger/) الخاصة بـ n8n.
