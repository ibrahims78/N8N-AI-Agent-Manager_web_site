# عقدة BambooHR (نظام إدارة الموارد البشرية)

استخدم عقدة BambooHR لأتمتة سير العمل في BambooHR، وتكامل BambooHR مع التطبيقات الأخرى. يدعم n8n بشكل مدمج مجموعة واسعة من ميزات BambooHR، بما في ذلك إنشاء تقارير الشركة ومستندات الموظفين والملفات وحذفها وتنزيلها واستردادها.

ستجد في هذه الصفحة قائمة بالعمليات التي تدعمها عقدة BambooHR وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد BambooHR](/integrations/builtin/credentials/bamboohr.md) للحصول على إرشادات حول إعداد المصادقة.

## العمليات

*   تقرير الشركة
    *   استرداد تقرير شركة
*   الموظف
    *   إنشاء موظف
    *   استرداد موظف
    *   استرداد جميع الموظفين
    *   تحديث موظف
*   مستند الموظف
    *   حذف مستند موظف
    *   تنزيل مستند موظف
    *   استرداد جميع مستندات الموظفين
    *   تحديث مستند موظف
    *   رفع مستند موظف
*   الملف
    *   حذف ملف شركة
    *   تنزيل ملف شركة
    *   استرداد جميع ملفات الشركة
    *   تحديث ملف شركة
    *   رفع ملف شركة
