# عقدة Shopify

استخدم عقدة Shopify لأتمتة سير العمل في Shopify، ودمج Shopify مع تطبيقات أخرى. تدعم n8n بشكل مدمج مجموعة واسعة من ميزات Shopify، بما في ذلك إنشاء الطلبات والمنتجات وتحديثها وحذفها والحصول عليها.

في هذه الصفحة، ستجد قائمة بالعمليات التي تدعمها عقدة Shopify وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد Shopify](/integrations/builtin/credentials/shopify.md) للحصول على إرشادات حول إعداد المصادقة.

## العمليات

*   الطلب
    *   إنشاء طلب
    *   حذف طلب
    *   الحصول على طلب
    *   الحصول على جميع الطلبات
    *   تحديث طلب
*   المنتج
    *   إنشاء منتج
    *   حذف منتج
    *   الحصول على منتج
    *   الحصول على جميع المنتجات
    *   تحديث منتج

## ماذا تفعل إذا كانت عمليتك غير مدعومة

إذا كانت هذه العقدة لا تدعم العملية التي ترغب في تنفيذها، يمكنك استخدام [عقدة طلب HTTP (HTTP Request)](/integrations/builtin/core-nodes/n8n-nodes-base.httprequest/index.md) لاستدعاء واجهة برمجة تطبيقات الخدمة.

يمكنك استخدام بيانات الاعتماد التي أنشأتها لهذه الخدمة في عقدة طلب HTTP (HTTP Request):

1.  في عقدة طلب HTTP (HTTP Request)، حدد **المصادقة** > **نوع بيانات الاعتماد المُحددة مسبقًا**.
2.  حدد الخدمة التي ترغب في الاتصال بها.
3.  حدد بيانات الاعتماد الخاصة بك.

ارجع إلى [عمليات API المخصصة](/integrations/custom-operations.md) لمزيد من المعلومات.
