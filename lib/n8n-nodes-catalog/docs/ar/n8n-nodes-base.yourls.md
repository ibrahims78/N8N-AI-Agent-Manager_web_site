# عقدة Yourls

استخدم عقدة Yourls لأتمتة سير العمل في Yourls، ودمج Yourls مع تطبيقات أخرى. تدعم n8n بشكل مدمج مجموعة واسعة من ميزات Yourls، بما في ذلك توسيع واختصار عناوين URL.

في هذه الصفحة، ستجد قائمة بالعمليات التي تدعمها عقدة Yourls وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد Yourls](/integrations/builtin/credentials/yourls.md) للحصول على إرشادات حول إعداد المصادقة.

## العمليات

* عنوان URL
    * توسيع عنوان URL
    * اختصار عنوان URL
    * الحصول على إحصائيات حول عنوان URL قصير واحد
