# عقدة Keap المُحفِّز

[Keap](https://keap.com/) هي منصة تسويق ومبيعات عبر البريد الإلكتروني للشركات الصغيرة، تتضمن منتجات لإدارة وتحسين دورة حياة العميل، وإدارة علاقات العملاء، وأتمتة التسويق، والتقاط العملاء المحتملين، والتجارة الإلكترونية.

> **بيانات الاعتماد**
>
> يمكن العثور على معلومات المصادقة لهذه العقدة [هنا](/integrations/builtin/credentials/keap.md).

> **أمثلة وقوالب**
>
> للحصول على أمثلة الاستخدام والقوالب لمساعدتك على البدء، ارجع إلى صفحة [تكاملات Keap Trigger](https://n8n.io/integrations/keap-trigger/) الخاصة بـ n8n.
