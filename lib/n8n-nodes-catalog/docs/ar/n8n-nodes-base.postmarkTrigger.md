# عقدة Postmark المُحفِّز

تساعد [Postmark](https://postmarkapp.com) في تسليم وتتبع رسائل البريد الإلكتروني للتطبيقات. يمكنك تتبع الإحصائيات مثل عدد رسائل البريد الإلكتروني المرسلة أو المعالجة، وعمليات الفتح، والارتدادات، وشكاوى البريد المزعج.

> **بيانات الاعتماد**
>
> يمكنك العثور على معلومات المصادقة لهذه العقدة [هنا](/integrations/builtin/credentials/postmark.md).

> **أمثلة وقوالب**
>
> للحصول على أمثلة الاستخدام والقوالب لمساعدتك على البدء، ارجع إلى صفحة [تكاملات Postmark المُحفِّز](https://n8n.io/integrations/postmark-trigger/) الخاصة بـ n8n.
