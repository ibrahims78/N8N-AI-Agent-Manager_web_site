# عقدة Bitwarden

استخدم عقدة Bitwarden (نظام إدارة كلمات المرور) لأتمتة سير العمل في Bitwarden، ودمج Bitwarden مع تطبيقات أخرى. يدعم n8n بشكل مدمج مجموعة واسعة من ميزات Bitwarden، بما في ذلك إنشاء المجموعات والأحداث والمجموعات والأعضاء واستردادها وحذفها وتحديثها.

ستجد في هذه الصفحة قائمة بالعمليات التي تدعمها عقدة Bitwarden وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد Bitwarden](/integrations/builtin/credentials/bitwarden.md) للحصول على إرشادات حول إعداد المصادقة.
