# عقدة QuickChart

استخدم عقدة QuickChart لأتمتة سير العمل في QuickChart، ودمج QuickChart مع تطبيقات أخرى. تدعم n8n بشكل مدمج مجموعة واسعة من أنواع المخططات (chart types) في QuickChart، بما في ذلك المخططات الشريطية (bar)، والدائرية المجوفة (doughnut)، والخطية (line)، والدائرية (pie)، والقطبية (polar).

في هذه الصفحة، ستجد قائمة بالعمليات التي تدعمها عقدة QuickChart وروابط لموارد إضافية.

> **يمكن استخدام هذه العقدة كأداة ذكاء اصطناعي**
>
> يمكن استخدام هذه العقدة لتعزيز قدرات وكيل الذكاء الاصطناعي. عند استخدامها بهذه الطريقة، يمكن تعيين العديد من المعاملات تلقائيًا، أو بمعلومات موجهة بواسطة الذكاء الاصطناعي - اكتشف المزيد في [توثيق معاملات أداة الذكاء الاصطناعي](/advanced-ai/examples/using-the-fromai-function.md).
