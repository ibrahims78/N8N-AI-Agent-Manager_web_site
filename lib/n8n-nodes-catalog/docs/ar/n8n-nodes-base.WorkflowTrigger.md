# عقدة Workflow Trigger

يتم تحفيز عقدة Workflow Trigger (مُحفِّز سير العمل) عند تحديث سير عمل أو تفعيله.

> **مُهمَلة**
>
> قامت n8n بإهمال عقدة Workflow Trigger ونقلت وظائفها إلى [عقدة n8n Trigger](/integrations/builtin/core-nodes/n8n-nodes-base.n8ntrigger.md).

> **ضع في اعتبارك**
>
> إذا كنت ترغب في استخدام عقدة Workflow Trigger لسير عمل، أضف العقدة إلى سير العمل. لا يلزم إنشاء سير عمل منفصل.

يتم تحفيز عقدة Workflow Trigger لسير العمل الذي تُضاف إليه. يمكنك استخدام عقدة Workflow Trigger لتحفيز سير عمل للإبلاغ عن حالة سير العمل.

## معاملات العقدة

تتضمن العقدة معاملاً واحداً لتحديد **الأحداث** التي يجب أن تحفزها. اختر من هذه الأحداث:

-   **Active Workflow Updated**: إذا حددت هذا الحدث، يتم تحفيز العقدة عند تحديث سير العمل هذا.
-   **Workflow Activated**: إذا حددت هذا الحدث، يتم تحفيز العقدة عند تفعيل سير العمل هذا.

يمكنك تحديد واحد أو كلا هذين الحدثين.
