# عقدة Microsoft To Do

استخدم عقدة Microsoft To Do لأتمتة سير العمل في Microsoft To Do، ودمج Microsoft To Do مع تطبيقات أخرى. يدعم n8n مجموعة واسعة من ميزات Microsoft To Do بشكل مدمج، بما في ذلك إنشاء الموارد المرتبطة والقوائم والمهام وتحديثها وحذفها والحصول عليها.

ستجد في هذه الصفحة قائمة بالعمليات التي تدعمها عقدة Microsoft To Do وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد Microsoft](/integrations/builtin/credentials/microsoft.md) للحصول على إرشادات حول إعداد المصادقة.

> **دعم السحابة الحكومية**
>
> إذا كنت تستخدم مستأجر سحابة حكومية (حكومة الولايات المتحدة، وزارة الدفاع الأمريكية، أو الصين)، فتأكد من تحديد **Microsoft Graph API Base URL** المناسب في إعدادات بيانات اعتماد Microsoft الخاصة بك.

> **يمكن استخدام هذه العقدة كأداة ذكاء اصطناعي**
>
> يمكن استخدام هذه العقدة لتعزيز قدرات وكيل الذكاء الاصطناعي. عند استخدامها بهذه الطريقة، يمكن تعيين العديد من المعاملات تلقائيًا، أو باستخدام معلومات موجهة بواسطة الذكاء الاصطناعي - اكتشف المزيد في [توثيق معاملات أداة الذكاء الاصطناعي](/advanced-ai/examples/using_the_fromai_function.md).

## ماذا تفعل إذا كانت عمليتك غير مدعومة

إذا كانت هذه العقدة لا تدعم العملية التي ترغب في تنفيذها، يمكنك استخدام [عقدة HTTP Request](/integrations/builtin/core-nodes/n8n-nodes-base.httprequest/index.md) لاستدعاء واجهة برمجة تطبيقات الخدمة.

يمكنك استخدام بيانات الاعتماد التي أنشأتها لهذه الخدمة في عقدة HTTP Request:

1.  في عقدة HTTP Request، حدد **Authentication** > **Predefined Credential Type**.
2.  حدد الخدمة التي ترغب في الاتصال بها.
3.  حدد بيانات الاعتماد الخاصة بك.

ارجع إلى [عمليات API المخصصة](/integrations/custom-operations.md) لمزيد من المعلومات.
