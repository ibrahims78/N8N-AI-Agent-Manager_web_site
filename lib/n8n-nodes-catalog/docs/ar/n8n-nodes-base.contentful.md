# عقدة Contentful

استخدم عقدة Contentful لأتمتة سير العمل في Contentful، ودمج Contentful مع تطبيقات أخرى. يدعم n8n بشكل مدمج مجموعة واسعة من ميزات Contentful، بما في ذلك استرداد الأصول وأنواع المحتوى والمدخلات واللغات والمساحات.

في هذه الصفحة، ستجد قائمة بالعمليات التي تدعمها عقدة Contentful وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد Contentful](/integrations/builtin/credentials/contentful.md) للحصول على إرشادات حول إعداد المصادقة.
