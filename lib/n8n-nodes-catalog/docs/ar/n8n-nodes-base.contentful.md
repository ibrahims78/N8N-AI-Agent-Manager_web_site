# Contentful (عقدة Contentful)

استخدم عقدة Contentful (عقدة Contentful) لأتمتة العمل في Contentful، ولدمج Contentful مع تطبيقات أخرى. يملك n8n دعماً مدمجاً لمجموعة واسعة من ميزات Contentful، بما في ذلك الحصول على الأصول، وأنواع المحتوى، والإدخالات، والإعدادات اللغوية، والفضاء.

في هذه الصفحة، ستجد قائمة بالعمليات التي تدعمها العقدة Contentful وروابط إلى موارد إضافية.
