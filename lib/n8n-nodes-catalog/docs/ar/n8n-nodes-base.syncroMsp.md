# عقدة SyncroMSP

استخدم عقدة SyncroMSP (نظام إدارة الخدمات المدارة) لأتمتة سير العمل في SyncroMSP، ودمج SyncroMSP مع تطبيقات أخرى. يدعم n8n مجموعة واسعة من ميزات SyncroMSP بشكل مدمج، بما في ذلك إنشاء وحذف العملاء الجدد، التذاكر، وجهات الاتصال.

ستجد في هذه الصفحة قائمة بالعمليات التي تدعمها عقدة SyncroMSP وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد SyncroMSP](/integrations/builtin/credentials/syncromsp.md) للحصول على إرشادات حول إعداد المصادقة.

    *   حذف جهة اتصال

    *   تحديث جهة اتصال

    *   إنشاء عميل جديد
    *   حذف عميل

*   RMM

    *   حذف تنبيه RMM
    *   استرداد تنبيه RMM
    *   استرداد جميع تنبيهات RMM
    *   كتم تنبيه RMM
*   التذكرة
    *   إنشاء تذكرة جديدة
    *   حذف تذكرة

    *   تحديث تذكرة
