# الفرز

استخدم عقدة الفرز (Sort) لتنظيم قوائم العناصر بترتيب مرغوب، أو لإنشاء تحديد عشوائي.

> **سلوك فرز المصفوفة**
>
> تستخدم عملية الفرز (Sort) عملية JavaScript الافتراضية حيث يتم تحويل العناصر المراد فرزها إلى سلاسل نصية ومقارنة قيمها. ارجع إلى [دليل Mozilla لفرز المصفوفات](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/sort) لمعرفة المزيد.

## معاملات العقدة

قم بتكوين هذه العقدة باستخدام المعامل **Type**.

استخدم القائمة المنسدلة لتحديد كيفية إدخال الفرز من هذه الخيارات.

### بسيط

يُجري فرزًا تصاعديًا أو تنازليًا باستخدام الحقول المحددة.

عند تحديد هذا **Type**:

*   استخدم زر **Add Field To Sort By** لإدخال **Field Name**.
*   حدد ما إذا كنت تريد استخدام ترتيب **Ascending** (تصاعدي) أو **Descending** (تنازلي).

#### خيارات بسيطة

عند تحديد **Simple** كـ **Type**، يتوفر لديك خيار **Disable Dot Notation**. بشكل افتراضي، تُمكّن n8n تدوين النقطة (dot notation) للإشارة إلى الحقول الفرعية بالصيغة `parent.child`. استخدم هذا الخيار لتعطيل تدوين النقطة (عند تشغيله) أو للاستمرار في استخدامه (عند إيقاف تشغيله).

### عشوائي

يُنشئ ترتيبًا عشوائيًا في القائمة.

### كود

أدخل كود JavaScript مخصصًا لإجراء عملية الفرز. هذا خيار جيد إذا لم يلبي الفرز البسيط احتياجاتك.

أدخل كود JavaScript المخصص الخاص بك في حقل الإدخال **Code**.
