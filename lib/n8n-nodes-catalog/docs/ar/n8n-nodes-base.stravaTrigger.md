# عقدة مُحفِّز Strava (سترافا)

[Strava](https://www.strava.com/) هي خدمة إنترنت لتتبع التمارين البشرية تتضمن ميزات الشبكات الاجتماعية.

> **بيانات الاعتماد**
>
> يمكنك العثور على معلومات المصادقة لهذه العقدة [هنا](/integrations/builtin/credentials/strava.md).

> **أمثلة وقوالب**
>
> لأمثلة الاستخدام والقوالب لمساعدتك على البدء، ارجع إلى صفحة [تكاملات مُحفِّز Strava](https://n8n.io/integrations/strava-trigger/) الخاصة بـ n8n.
