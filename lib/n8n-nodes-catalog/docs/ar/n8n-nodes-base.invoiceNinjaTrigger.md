# العقدة المُحفِّزة Invoice Ninja

[Invoice Ninja](https://www.invoiceninja.com/) هو تطبيق مجاني مفتوح المصدر للفوترة عبر الإنترنت للمستقلين والشركات. يوفر الفوترة، المدفوعات، تتبع المصروفات، والمهام الزمنية.

> **بيانات الاعتماد**
>
> يمكن العثور على معلومات المصادقة لهذه العقدة [هنا](/integrations/builtin/credentials/invoiceninja.md).

> **أمثلة وقوالب**
>
> للحصول على أمثلة الاستخدام والقوالب لمساعدتك على البدء، ارجع إلى صفحة [تكاملات Invoice Ninja Trigger](https://n8n.io/integrations/invoice-ninja-trigger/) الخاصة بـ n8n.
