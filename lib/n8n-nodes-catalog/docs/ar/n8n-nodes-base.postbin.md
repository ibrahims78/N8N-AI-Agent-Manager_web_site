# عقدة PostBin

PostBin هي خدمة تساعدك على اختبار عملاء API وويب هوكس. استخدم عقدة PostBin لأتمتة العمل في PostBin، ودمج PostBin مع تطبيقات أخرى. يدعم n8n بشكل مدمج مجموعة واسعة من ميزات PostBin، بما في ذلك إنشاء وحذف الحاويات، واستلام وإرسال الطلبات.

في هذه الصفحة، ستجد قائمة بالعمليات التي تدعمها عقدة PostBin، وروابط لموارد إضافية.

## العمليات

*   حاوية
    *   إنشاء
    *   استلام
    *   حذف
*   طلب
    *   استلام
    *   إزالة الأول
    *   إرسال

## إرسال الطلبات

لإرسال الطلبات إلى حاوية PostBin:

1.  انتقل إلى [PostBin](https://www.toptal.com/developers/postbin/) واتبع الخطوات لإنشاء حاوية جديدة. يمنحك PostBin عنوان URL فريدًا، يتضمن معرف الحاوية.
2.  في عقدة PostBin، حدد المورد **Request**.
3.  اختر نوع **العملية** التي ترغب في تنفيذها.
4.  أدخل معرف الحاوية الخاص بك في **Bin ID**.

## إنشاء وإدارة الحاويات

يمكنك إنشاء وإدارة حاويات PostBin باستخدام عقدة PostBin.

1.  في **Resource**، حدد **Bin**.
2.  اختر **عملية**. يمكنك إنشاء أو حذف أو استلام حاوية.
