# عقدة AWS S3

استخدم عقدة AWS S3 (خدمة التخزين البسيط من أمازون) لأتمتة سير العمل في AWS S3، وتكامل AWS S3 مع التطبيقات الأخرى. تدعم n8n بشكل مدمج مجموعة واسعة من ميزات AWS S3، بما في ذلك إنشاء وحذف المجموعات (buckets)، ونسخ وتنزيل الملفات، بالإضافة إلى الحصول على المجلدات.

ستجد في هذه الصفحة قائمة بالعمليات التي تدعمها عقدة AWS S3 وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد AWS](/integrations/builtin/credentials/aws.md) للحصول على إرشادات حول إعداد المصادقة.

> **يمكن استخدام هذه العقدة كأداة ذكاء اصطناعي**
>
> يمكن استخدام هذه العقدة لتعزيز قدرات وكيل الذكاء الاصطناعي. عند استخدامها بهذه الطريقة، يمكن تعيين العديد من المعاملات تلقائيًا، أو باستخدام معلومات موجهة بواسطة الذكاء الاصطناعي - اكتشف المزيد في [وثائق معاملات أداة الذكاء الاصطناعي](/advanced-ai/examples/using-the-fromai-function.md).

## العمليات

*   المجموعة (Bucket)
    *   إنشاء مجموعة
    *   حذف مجموعة
    *   الحصول على جميع المجموعات
    *   البحث داخل مجموعة
*   الملف
    *   نسخ ملف
    *   حذف ملف
    *   تنزيل ملف
    *   الحصول على جميع الملفات
    *   رفع ملف
*   المجلد
    *   إنشاء مجلد
    *   حذف مجلد
    *   الحصول على جميع المجلدات

## ماذا تفعل إذا كانت عمليتك غير مدعومة

إذا كانت هذه العقدة لا تدعم العملية التي ترغب في القيام بها، يمكنك استخدام [عقدة طلب HTTP](/integrations/builtin/core-nodes/n8n-nodes-base.httprequest/index.md) لاستدعاء واجهة برمجة تطبيقات الخدمة.

يمكنك استخدام بيانات الاعتماد التي أنشأتها لهذه الخدمة في عقدة طلب HTTP:

1.  في عقدة طلب HTTP، حدد **Authentication** > **Predefined Credential Type**.
2.  حدد الخدمة التي تريد الاتصال بها.
3.  حدد بيانات الاعتماد الخاصة بك.

ارجع إلى [عمليات API المخصصة](/integrations/custom-operations.md) لمزيد من المعلومات.
