# عقدة Metabase

استخدم عقدة Metabase لأتمتة سير العمل في Metabase، ودمج Metabase مع التطبيقات الأخرى. تدعم n8n بشكل مدمج مجموعة واسعة من ميزات Metabase، بما في ذلك إضافة وتلقي التنبيهات وقواعد البيانات والمقاييس والأسئلة.

في هذه الصفحة، ستجد قائمة بالعمليات التي تدعمها عقدة Metabase وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد Metabase](/integrations/builtin/credentials/metabase.md) للحصول على إرشادات حول إعداد المصادقة.

## ماذا تفعل إذا كانت عمليتك غير مدعومة

إذا كانت هذه العقدة لا تدعم العملية التي ترغب في تنفيذها، يمكنك استخدام [عقدة HTTP Request](/integrations/builtin/core-nodes/n8n-nodes-base.httprequest/index.md) لاستدعاء واجهة برمجة تطبيقات الخدمة.

يمكنك استخدام بيانات الاعتماد التي أنشأتها لهذه الخدمة في عقدة HTTP Request:

1.  في عقدة HTTP Request، حدد **المصادقة** > **نوع بيانات الاعتماد المُعرّف مسبقًا**.
2.  حدد الخدمة التي ترغب في الاتصال بها.
3.  حدد بيانات الاعتماد الخاصة بك.

ارجع إلى [عمليات API المخصصة](/integrations/custom-operations.md) لمزيد من المعلومات.
