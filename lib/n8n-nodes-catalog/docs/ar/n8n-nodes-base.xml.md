# XML

استخدم عقدة XML لتحويل البيانات من وإلى XML.

> **الملفات الثنائية**
>
> إذا كان ملف XML الخاص بك موجودًا ضمن ملف ثنائي، فاستخدم عقدة [Extract from File](/integrations/builtin/core-nodes/n8n-nodes-base.extractfromfile.md) لتحويله إلى نص أولاً.

## معاملات العقدة

-   **الوضع (Mode)**: التنسيق الذي يجب تحويل البيانات منه وإليه.
    -   **من JSON إلى XML (JSON to XML)**: يحول البيانات من JSON إلى XML.
    -   **من XML إلى JSON (XML to JSON)**: يحول البيانات من XML إلى JSON.
-   **اسم الخاصية (Property Name)**: أدخل اسم الخاصية التي تحتوي على البيانات المراد تحويلها.

## خيارات العقدة

تتوفر هذه الخيارات بغض النظر عن **الوضع (Mode)** الذي تحدده:

-   **مفتاح السمة (Attribute Key)**: أدخل البادئة المستخدمة للوصول إلى السمات. القيمة الافتراضية هي `$`.
-   **مفتاح الحرف (Character Key)**: أدخل البادئة المستخدمة للوصول إلى محتوى الحرف. القيمة الافتراضية هي `_`.

تعتمد جميع الخيارات الأخرى على **الوضع (Mode)** المحدد.

### خيارات التحويل من JSON إلى XML

تظهر هذه الخيارات فقط إذا حددت **من JSON إلى XML (JSON to XML)** كـ **الوضع (Mode)**:

-   **السماح بالأحرف البديلة (Allow Surrogate Chars)**: حدد ما إذا كنت تريد السماح باستخدام أحرف من كتل Unicode البديلة (مفعل) أم لا (معطل).
-   **بيانات CDATA (Cdata)**: حدد ما إذا كنت تريد تضمين العقد النصية في `<![CDATA[ ... ]]>` بدلاً من الهروب عند الحاجة (مفعل) أم لا (معطل).
    *   لا يؤدي تفعيل هذا الخيار إلى إضافة `<![CDATA[ ... ]]>` إذا لم يكن مطلوبًا.
-   **بدون رأس (Headless)**: حدد ما إذا كنت تريد حذف ترويسة XML (مفعل) أم تضمينها (معطل).
-   **اسم الجذر (Root Name)**: أدخل اسم العنصر الجذر المراد استخدامه.

### خيارات التحويل من XML إلى JSON

تظهر هذه الخيارات فقط إذا حددت **من XML إلى JSON (XML to JSON)** كـ **الوضع (Mode)**:

-   **مصفوفة صريحة (Explicit Array)**: حدد ما إذا كنت تريد وضع العقد الفرعية في مصفوفة (مفعل) أم إنشاء مصفوفة فقط إذا كان هناك أكثر من عقدة فرعية واحدة (معطل).
-   **جذر صريح (Explicit Root)**: حدد ما إذا كنت تريد الحصول على العقدة الجذرية في الكائن الناتج (مفعل) أم لا (معطل).
-   **تجاهل السمات (Ignore Attributes)**: حدد ما إذا كنت تريد تجاهل جميع سمات XML وإنشاء عقد نصية فقط (مفعل) أم لا (معطل).
-   **دمج السمات (Merge Attributes)**: حدد ما إذا كنت تريد دمج السمات والعناصر الفرعية كخصائص للعنصر الأصل (مفعل) أم سمات مفتاحية لكائن سمة فرعي (معطل). يتم تجاهل هذا الخيار إذا تم تفعيل **تجاهل السمات (Ignore Attributes)**.
-   **تطبيع (Normalize)**: حدد ما إذا كنت تريد إزالة المسافات البيضاء داخل العقد النصية (مفعل) أم عدم إزالتها (معطل).
-   **تطبيع الوسوم (Normalize Tags)**: حدد ما إذا كنت تريد تطبيع جميع أسماء الوسوم إلى أحرف صغيرة (مفعل) أم الاحتفاظ بأسماء الوسوم كما هي (معطل).
-   **اقتطاع (Trim)**: حدد ما إذا كنت تريد اقتطاع المسافات البيضاء في بداية ونهاية العقد النصية (مفعل) أم ترك المسافات البيضاء كما هي (معطل).
