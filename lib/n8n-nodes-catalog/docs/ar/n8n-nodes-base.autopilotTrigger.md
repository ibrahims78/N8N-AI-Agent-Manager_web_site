# عقدة مُحفِّز Autopilot

[Autopilot](https://www.autopilothq.com/) هو برنامج تسويق مرئي يتيح لك أتمتة وتسويقك وتخصيصه عبر رحلة العميل بأكملها.

> **بيانات الاعتماد**
>
> يمكنك العثور على معلومات المصادقة لهذه العقدة [هنا](/integrations/builtin/credentials/autopilot.md).

> **أمثلة وقوالب**
>
> للحصول على أمثلة الاستخدام والقوالب لمساعدتك على البدء، ارجع إلى صفحة [تكاملات مُحفِّز Autopilot](https://n8n.io/integrations/autopilot-trigger/) الخاصة بـ n8n.

## الأحداث

- إضافة جهة اتصال
- إضافة جهة اتصال إلى قائمة
- دخول جهة اتصال إلى شريحة
- مغادرة جهة اتصال لشريحة
- إزالة جهة اتصال من قائمة
- إلغاء اشتراك جهة اتصال
- تحديث جهة اتصال
