# عقدة Microsoft SharePoint

استخدم عقدة Microsoft SharePoint لأتمتة سير العمل في Microsoft SharePoint ودمج Microsoft SharePoint مع تطبيقات أخرى. تدعم n8n مجموعة واسعة من ميزات Microsoft SharePoint بشكل مدمج، والتي تشمل تنزيل الملفات وتحميلها وتحديثها، وإدارة العناصر في قائمة، والحصول على القوائم وعناصر القوائم.

في هذه الصفحة، ستجد قائمة بالعمليات التي تدعمها عقدة Microsoft SharePoint، وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> يمكنك العثور على معلومات المصادقة لهذه العقدة [هنا](/integrations/builtin/credentials/microsoft.md).

## العمليات

<!-- To avoid warning about "many" -->
<!-- vale from-write-good.Weasel = NO -->
* **الملف**:
	* تنزيل: تنزيل ملف.
	* تحديث: تحديث ملف.
	* تحميل: تحميل ملف موجود.
* **العنصر**:
	* إنشاء: إنشاء عنصر في قائمة موجودة.
	* إنشاء أو تحديث: إنشاء عنصر جديد، أو تحديث العنصر الحالي إذا كان موجودًا بالفعل (upsert).
	* حذف: حذف عنصر من قائمة.
	* الحصول: استرداد عنصر من قائمة.
	* الحصول على العديد: الحصول على عناصر محددة في قائمة أو سرد العديد من العناصر.
	* تحديث: تحديث عنصر في قائمة موجودة.
* **القائمة**:
	* الحصول: استرداد تفاصيل قائمة واحدة.
	* الحصول على العديد: استرداد قائمة بالقوائم.
