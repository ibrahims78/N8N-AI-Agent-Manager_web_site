# عقدة مُحمِّل مستندات GitHub

> **مهملة**
>
> هذه العقدة مهملة، وستتم إزالتها في إصدار مستقبلي.

استخدم عقدة مُحمِّل مستندات GitHub لتحميل البيانات من مستودع GitHub لمخازن المتجهات [vector stores](/glossary.md#ai-vector-store) أو التلخيص.

في هذه الصفحة، ستجد معاملات العقدة لعقدة مُحمِّل مستندات GitHub، وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> يمكنك العثور على معلومات المصادقة [authentication](/integrations/builtin/credentials/github.md) لهذه العقدة هنا. لا تدعم هذه العقدة OAuth للمصادقة.

> **حل المعاملات في العقد الفرعية**
>
> تتصرف العقد الفرعية بشكل مختلف عن العقد الأخرى عند معالجة عناصر متعددة باستخدام تعبير.
>
> تقبل معظم العقد، بما في ذلك العقد الجذرية، أي عدد من العناصر كمدخلات، وتعالج هذه العناصر، وتُخرج النتائج. يمكنك استخدام التعبيرات للإشارة إلى عناصر المدخلات، وتقوم العقدة بحل التعبير لكل عنصر على حدة. على سبيل المثال، بالنظر إلى مدخلات من خمس قيم `name`، يتم حل التعبير `` لكل اسم على حدة.
>
> في العقد الفرعية، يتم حل التعبير دائمًا للعنصر الأول. على سبيل المثال، بالنظر إلى مدخلات من خمس قيم `name`، يتم حل التعبير `` دائمًا للاسم الأول.

## معاملات العقدة

*   **تقسيم النص**: اختر من:
    *   **بسيط**: يستخدم مُقسِّم النص الحرفي التكراري [Recursive Character Text Splitter](/integrations/builtin/cluster-nodes/sub-nodes/n8n-nodes-langchain.textsplitterrecursivecharactertextsplitter.md) بحجم جزء (chunk size) يبلغ 1000 وتداخل (overlap) يبلغ 200.
    *   **مخصص**: يسمح لك بربط مُقسِّم نص من اختيارك.
*   **رابط المستودع**: أدخل عنوان URL لمستودع GitHub الخاص بك.
*   **الفرع**: أدخل اسم الفرع المراد استخدامه.

## خيارات العقدة

*   **تكراري**: حدد ما إذا كنت تريد تضمين المجلدات الفرعية والملفات (مفعل) أم لا (معطل).
*   **مسارات التجاهل**: أدخل الدلائل المراد تجاهلها.
