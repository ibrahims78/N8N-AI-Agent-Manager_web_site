# عقدة Venafi TLS Protect Cloud Trigger

[Venafi](https://www.venafi.com/) هي شركة للأمن السيبراني تقدم خدمات لإدارة هوية الآلة. توفر حلولاً لإدارة وحماية الهويات لمجموعة واسعة من أنواع الآلات، مُوفِّرةً رؤية عالمية، وأتمتة لدورة الحياة، وذكاءً قابلاً للتنفيذ.

استخدم عقدة Venafi TLS Protect Cloud Trigger في n8n لبدء سير عمل في n8n استجابةً للأحداث في خدمة [Venafi TLS Protect السحابية](https://vaas.venafi.com/).

> **بيانات الاعتماد**
>
> يمكن العثور على معلومات المصادقة لهذه العقدة [هنا](/integrations/builtin/credentials/venafitlsprotectcloud.md).

> **أمثلة وقوالب**
>
> للحصول على أمثلة استخدام وقوالب لمساعدتك على البدء، ارجع إلى صفحة [تكاملات Venafi TLS Protect Cloud Trigger](https://n8n.io/integrations/venafi-tls-protect-cloud-trigger/) الخاصة بـ n8n.
