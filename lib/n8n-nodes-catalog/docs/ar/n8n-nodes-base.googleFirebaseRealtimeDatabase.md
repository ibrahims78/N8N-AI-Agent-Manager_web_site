# عقدة Google Cloud Realtime Database

استخدم عقدة Google Cloud Realtime Database (قاعدة بيانات Google Cloud في الوقت الفعلي) لأتمتة سير العمل في Google Cloud Realtime Database، ودمج Google Cloud Realtime Database مع تطبيقات أخرى. تدعم n8n بشكل مدمج مجموعة واسعة من ميزات Google Cloud Realtime Database، بما في ذلك كتابة البيانات في قواعد البيانات، وحذفها، واسترجاعها، وإلحاقها.

في هذه الصفحة، ستجد قائمة بالعمليات التي تدعمها عقدة Google Cloud Realtime Database وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد Google Cloud Realtime Database](/integrations/builtin/credentials/google/index.md) للحصول على إرشادات حول إعداد المصادقة.

> **يمكن استخدام هذه العقدة كأداة ذكاء اصطناعي**
>
> يمكن استخدام هذه العقدة لتعزيز قدرات وكيل ذكاء اصطناعي. عند استخدامها بهذه الطريقة، يمكن تعيين العديد من المعاملات تلقائيًا، أو بمعلومات موجهة بواسطة الذكاء الاصطناعي - اكتشف المزيد في [توثيق معاملات أداة الذكاء الاصطناعي](/advanced-ai/examples/using-the-fromai-function.md).

## العمليات

*   كتابة البيانات إلى قاعدة بيانات
*   حذف البيانات من قاعدة بيانات
*   استرجاع سجل من قاعدة بيانات
*   إلحاق بقائمة من البيانات
*   تحديث عنصر في قاعدة بيانات
