# العقدة المُحفِّزة Lemlist

[Lemlist](https://Lemlist.com) (منصة للتواصل عبر البريد الإلكتروني) هي منصة تتيح لك إنشاء صور ومقاطع فيديو مخصصة تلقائيًا وإرسال رسائل بريد إلكتروني باردة مخصصة.

> **بيانات الاعتماد**
>
> يمكنك العثور على معلومات المصادقة لهذه العقدة [هنا](/integrations/builtin/credentials/lemlist.md).

> **أمثلة وقوالب**
>
> للحصول على أمثلة الاستخدام والقوالب لمساعدتك على البدء، ارجع إلى صفحة [تكاملات Lemlist Trigger](https://n8n.io/integrations/lemlist-trigger/) الخاصة بـ n8n.
