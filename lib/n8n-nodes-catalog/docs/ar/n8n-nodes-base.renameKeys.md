# Rename Keys

استخدم العقدة Rename Keys (إعادة تسمية المفاتيح) لإعادة تسمية مفاتيح أزواج المفتاح-قيمة في n8n.

## Node parameters

يمكنك إعادة تسمية مفتاح واحد أو عدة مفاتيح باستخدام العقدة Rename Keys. حدد زر **Add new key** (إضافة مفتاح جديد) لإعادة تسمية مفتاح.

لكل مفتاح، أدخل:

-   **Current Key Name**: الاسم الحالي للمفتاح الذي تريد إعادة تسميته.
-   **New Key Name**: الاسم الجديد الذي تريد تعيينه للمفتاح.

## Node options

اختر ما إذا كنت تريد استخدام تعبير نمطي (Regex) لتحديد المفاتيح المراد إعادة تسميتها. لاستخدام هذا الخيار، يجب عليك أيضًا إدخال:

*   **Regular Expression**: التعبير النمطي الذي ترغب في استخدامه.
*   **Replace With**: أدخل الاسم الجديد الذي تريد تعيينه للمفتاح (المفاتيح) التي تطابق **Regular Expression**.
*   يمكنك أيضًا اختيار هذه الخيارات الخاصة بـ Regex:
    *   **Case Insensitive**: حدد ما إذا كان التعبير النمطي يجب أن يطابق حالة الأحرف (معطل) أو يكون غير حساس لحالة الأحرف (مفعل).
    *   **Max Depth**: أدخل أقصى عمق لاستبدال المفاتيح، باستخدام `-1` لغير محدود و`0` للمستوى الأعلى فقط.

> **Regex impacts**
>
> يمكن أن يؤثر استخدام تعبير نمطي على أي مفاتيح تطابق التعبير، بما في ذلك المفاتيح التي قمت بإعادة تسميتها بالفعل.
