# عقدة Mocean

استخدم عقدة Mocean (منصة اتصالات سحابية) لأتمتة سير العمل في Mocean، ودمج Mocean مع تطبيقات أخرى. تدعم n8n بشكل مدمج مجموعة واسعة من ميزات Mocean، بما في ذلك إرسال رسائل SMS والرسائل الصوتية.

في هذه الصفحة، ستجد قائمة بالعمليات التي تدعمها عقدة Mocean وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد Mocean](/integrations/builtin/credentials/mocean.md) للحصول على إرشادات حول إعداد المصادقة.

## العمليات

*   SMS
    *   إرسال رسالة SMS/صوتية
*   صوت
    *   إرسال رسالة SMS/صوتية
