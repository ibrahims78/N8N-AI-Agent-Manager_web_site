# بيانات التنفيذ

استخدم هذه العقدة لحفظ البيانات الوصفية لعمليات تنفيذ سير العمل. يمكنك بعد ذلك البحث باستخدام هذه البيانات في قائمة **التنفيذات**.

يمكنك استرداد بيانات التنفيذ المخصصة أثناء تنفيذ سير العمل باستخدام عقدة Code. ارجع إلى [بيانات التنفيذ المخصصة](/workflows/executions/custom-executions-data.md) لمزيد من المعلومات.

> **توفر الميزة**
>
> تتوفر بيانات التنفيذ المخصصة على:
>
> * السحابة: Pro, Enterprise
> * الاستضافة الذاتية: Enterprise, Community المسجلة

## العمليات

* حفظ بيانات التنفيذ للبحث

## البيانات المراد حفظها

أضف **حقلاً محفوظاً** لكل زوج مفتاح/قيمة من البيانات الوصفية التي ترغب في حفظها.

## القيود

تفرض عقدة Execution Data القيود التالية عند تخزين البيانات الوصفية للتنفيذ:

* `key`: يقتصر على 50 حرفاً
* `value`: يقتصر على 512 حرفاً

إذا تجاوز أي من `key` أو `value` القيود المذكورة أعلاه، يقوم n8n باقتطاعها إلى أقصى طول لها ويُخرج إدخال سجل.
