# عقدة Vero

استخدم عقدة Vero لأتمتة سير العمل في Vero ودمج Vero مع تطبيقات أخرى. تدعم n8n مجموعة واسعة من ميزات Vero بشكل مدمج، بما في ذلك إنشاء المستخدمين وحذفهم.

ستجد في هذه الصفحة قائمة بالعمليات التي تدعمها عقدة Vero وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد Vero](/integrations/builtin/credentials/vero.md) للحصول على إرشادات حول إعداد المصادقة.

## العمليات

*   المستخدم
    *   إنشاء أو تحديث ملف تعريف مستخدم
    *   تغيير مُعرِّف مستخدم
    *   إلغاء اشتراك مستخدم.
    *   إعادة اشتراك مستخدم.
    *   حذف مستخدم.
    *   إضافة وسم (tag) إلى ملف تعريف مستخدم.
    *   إزالة وسم (tag) من ملف تعريف مستخدم.
*   الحدث
    *   تتبع حدث لعميل معين
