# عقدة Cockpit

استخدم عقدة Cockpit لأتمتة سير العمل في Cockpit، ودمج Cockpit مع تطبيقات أخرى. تدعم n8n مجموعة واسعة من ميزات Cockpit بشكل مدمج، بما في ذلك إنشاء إدخال مجموعة، وتخزين البيانات من إرسال نموذج، واسترداد العناصر الفردية (singletons).

في هذه الصفحة، ستجد قائمة بالعمليات التي تدعمها عقدة Cockpit وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد Cockpit](/integrations/builtin/credentials/cockpit.md) للحصول على إرشادات حول إعداد المصادقة.

## العمليات

*   المجموعة (Collection)
    *   إنشاء إدخال مجموعة
    *   استرداد جميع إدخالات المجموعة
    *   تحديث إدخال مجموعة
*   النموذج (Form)
    *   تخزين البيانات من إرسال نموذج
*   العنصر الفردي (Singleton)
    *   استرداد عنصر فردي
