# عقدة Unleashed Software (برنامج إدارة المخزون)

استخدم عقدة Unleashed Software لأتمتة سير العمل في Unleashed Software، ودمج Unleashed Software مع تطبيقات أخرى. تدعم n8n مجموعة واسعة من ميزات Unleashed Software بشكل مدمج، بما في ذلك جلب طلبات المبيعات والمخزون المتوفر.

ستجد في هذه الصفحة قائمة بالعمليات التي تدعمها عقدة Unleashed Software وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد Unleashed Software](/integrations/builtin/credentials/unleashedsoftware.md) للحصول على إرشادات حول إعداد المصادقة.
