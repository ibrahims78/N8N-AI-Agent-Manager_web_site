# العقدة المُحفِّزة Typeform

[Typeform](https://www.typeform.com/) هي شركة برمجيات كخدمة عبر الإنترنت متخصصة في بناء النماذج والاستبيانات عبر الإنترنت. ينشئ برنامجها الرئيسي نماذج ديناميكية بناءً على احتياجات المستخدم.

> **بيانات الاعتماد**
>
> يمكنك العثور على معلومات المصادقة لهذه العقدة [هنا](/integrations/builtin/credentials/typeform.md).

> **أمثلة وقوالب**
>
> للحصول على أمثلة الاستخدام والقوالب لمساعدتك على البدء، ارجع إلى صفحة [تكاملات Typeform Trigger](https://n8n.io/integrations/typeform-trigger/) الخاصة بـ n8n.
