# عقدة Jina AI

استخدم عقدة Jina AI (Jina AI node) لأتمتة سير العمل في Jina AI ودمج Jina AI مع التطبيقات الأخرى. يدعم n8n مجموعة واسعة من ميزات Jina AI بشكل مدمج.

في هذه الصفحة، ستجد قائمة بالعمليات التي تدعمها عقدة Jina AI، وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> يمكنك العثور على معلومات المصادقة لهذه العقدة [هنا](/integrations/builtin/credentials/jinaai.md).

## العمليات

*   **القارئ (Reader)**:
    *   **القراءة (Read)**: يجلب المحتوى من عنوان URL ويحوله إلى تنسيقات نظيفة وملائمة لنموذج اللغة الكبير (LLM).
    *   **البحث (Search)**: يُجري بحث ويب باستخدام Jina AI ويُعيد أفضل النتائج بتنسيقات نظيفة وملائمة لنموذج اللغة الكبير (LLM).
*   **البحث (Research)**:
    *   **البحث العميق (Deep Research)**: يبحث في موضوع ويُنشئ تقرير بحث منظم.

## الموارد ذات الصلة

ارجع إلى [توثيق API الخاص بقارئ Jina AI](https://r.jina.ai/docs) و [توثيق API الخاص ببحث Jina AI](https://s.jina.ai/docs) لمزيد من المعلومات حول الخدمة.

## ماذا تفعل إذا كانت عمليتك غير مدعومة

إذا كانت هذه العقدة لا تدعم العملية التي ترغب في تنفيذها، يمكنك استخدام [عقدة طلب HTTP (HTTP Request node)](/integrations/builtin/core-nodes/n8n-nodes-base.httprequest/index.md) لاستدعاء API الخاص بالخدمة.

يمكنك استخدام بيانات الاعتماد التي أنشأتها لهذه الخدمة في عقدة طلب HTTP:

1.  في عقدة طلب HTTP، اختر **المصادقة (Authentication)** > **نوع بيانات الاعتماد المُعرّف مسبقًا (Predefined Credential Type)**.
2.  اختر الخدمة التي ترغب في الاتصال بها.
3.  اختر بيانات الاعتماد الخاصة بك.

ارجع إلى [عمليات API المخصصة](/integrations/custom-operations.md) لمزيد من المعلومات.
