# عقدة Freshservice

استخدم عقدة Freshservice لأتمتة سير العمل في Freshservice ودمج Freshservice مع تطبيقات أخرى. تدعم n8n بشكل مدمج مجموعة واسعة من ميزات Freshdesk، بما في ذلك إنشاء معلومات الوكلاء والأقسام وتحديثها وحذفها واستردادها.

ستجد في هذه الصفحة قائمة بالعمليات التي تدعمها عقدة Freshservice وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد Freshservice](/integrations/builtin/credentials/freshservice.md) للحصول على إرشادات حول إعداد المصادقة.
