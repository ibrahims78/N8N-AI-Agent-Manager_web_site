# عقدة GoToWebinar (GoToWebinar)

استخدم عقدة GoToWebinar لأتمتة سير العمل في GoToWebinar، ودمج GoToWebinar مع التطبيقات الأخرى. تدعم n8n بشكل مدمج مجموعة واسعة من ميزات GoToWebinar، بما في ذلك إنشاء الحضور والمنظمين والمسجلين واستردادهم وحذفهم.

في هذه الصفحة، ستجد قائمة بالعمليات التي تدعمها عقدة GoToWebinar وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد GoToWebinar](/integrations/builtin/credentials/gotowebinar.md) للحصول على إرشادات حول إعداد المصادقة.

## ماذا تفعل إذا كانت عمليتك غير مدعومة

إذا كانت هذه العقدة لا تدعم العملية التي ترغب في تنفيذها، يمكنك استخدام [عقدة HTTP Request (طلب HTTP)](/integrations/builtin/core-nodes/n8n-nodes-base.httprequest/index.md) لاستدعاء واجهة برمجة تطبيقات (API) للخدمة.

يمكنك استخدام بيانات الاعتماد التي أنشأتها لهذه الخدمة في عقدة HTTP Request:

1.  في عقدة HTTP Request، حدد **Authentication** > **Predefined Credential Type**.
2.  حدد الخدمة التي ترغب في الاتصال بها.
3.  حدد بيانات الاعتماد الخاصة بك.

ارجع إلى [عمليات API المخصصة](/integrations/custom-operations.md) لمزيد من المعلومات.
