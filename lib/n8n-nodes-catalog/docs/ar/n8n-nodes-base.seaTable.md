# عقدة SeaTable

استخدم عقدة SeaTable لأتمتة سير العمل في SeaTable، ودمج SeaTable مع تطبيقات أخرى. تدعم n8n بشكل مدمج مجموعة واسعة من ميزات SeaTable، بما في ذلك إنشاء الصفوف وتحديثها وحذفها واستردادها.

في هذه الصفحة، ستجد قائمة بالعمليات التي تدعمها عقدة SeaTable وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد SeaTable](/integrations/builtin/credentials/seatable.md) للحصول على إرشادات حول إعداد المصادقة.

    *   تحديث
