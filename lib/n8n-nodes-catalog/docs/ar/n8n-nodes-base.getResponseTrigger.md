# العقدة المُحفِّزة GetResponse

[GetResponse](https://www.getresponse.com/) هي منصة عبر الإنترنت تقدم برنامج تسويق عبر البريد الإلكتروني، ومنشئ صفحات هبوط، واستضافة ندوات عبر الويب، وأكثر من ذلك بكثير.

> **بيانات الاعتماد**
>
> يمكنك العثور على معلومات المصادقة لهذه العقدة [هنا](/integrations/builtin/credentials/getresponse.md).

> **أمثلة وقوالب**
>
> للحصول على أمثلة الاستخدام والقوالب لمساعدتك على البدء، ارجع إلى صفحة [تكاملات GetResponse Trigger](https://n8n.io/integrations/getresponse-trigger/) الخاصة بـ n8n.

## الأحداث

-   تلقي إشعارات عند اشتراك عميل في قائمة
-   تلقي إشعارات عند إلغاء اشتراك عميل من قائمة
-   تلقي إشعارات عند فتح بريد إلكتروني
-   تلقي إشعارات عند النقر على بريد إلكتروني
-   تلقي إشعارات عند إرسال استبيان
