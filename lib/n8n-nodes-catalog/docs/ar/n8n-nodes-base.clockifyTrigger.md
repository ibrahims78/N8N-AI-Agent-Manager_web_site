# العقدة المُحفِّزة Clockify

[Clockify](https://clockify.me/) هو تطبيق مجاني لتتبع الوقت والجداول الزمنية لتتبع ساعات العمل عبر المشاريع.

> **بيانات الاعتماد**
>
> يمكنك العثور على معلومات المصادقة لهذه العقدة [هنا](/integrations/builtin/credentials/clockify.md).

> **أمثلة وقوالب**
>
> للحصول على أمثلة الاستخدام والقوالب لمساعدتك على البدء، ارجع إلى صفحة [تكاملات](https://n8n.io/integrations/clockify-trigger/) Clockify Trigger الخاصة بـ n8n.

تستخدم هذه العقدة إعداد المنطقة الزمنية لسير العمل لتحديد نطاق وقت بدء إدخالات الوقت. قم بتكوين المنطقة الزمنية في [إعدادات سير العمل](/workflows/settings.md) الخاصة بك إذا كنت تريد أن تسترد هذه العقدة المُحفِّزة إدخالات الوقت الصحيحة.
