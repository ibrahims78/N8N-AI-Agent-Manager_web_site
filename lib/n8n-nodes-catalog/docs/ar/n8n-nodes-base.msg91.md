# عقدة MSG91

استخدم عقدة MSG91 لأتمتة سير العمل في MSG91، ودمج MSG91 مع تطبيقات أخرى. تدعم n8n إرسال الرسائل القصيرة (SMS) باستخدام MSG91.

في هذه الصفحة، ستجد قائمة بالعمليات التي تدعمها عقدة MSG91 وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد MSG91](/integrations/builtin/credentials/msg91.md) للحصول على إرشادات حول إعداد المصادقة.

## العمليات

* الرسائل القصيرة (SMS)
    * إرسال الرسائل القصيرة (SMS)

## العثور على مُعرّف المُرسِل الخاص بك

1.  سجّل الدخول إلى لوحة تحكم MSG91 الخاصة بك.
2.  حدد **مُعرّف المُرسِل (Sender Id)** في اللوحة اليسرى.
3.  إذا لم يكن لديك واحد بالفعل، حدد **إضافة مُعرّف مُرسِل (Add Sender Id +)**، املأ التفاصيل، ثم حدد **حفظ مُعرّف المُرسِل (Save Sender Id)**.
