# عقدة مُحفِّز Help Scout

Help Scout (برنامج مكتب مساعدة) هو برنامج مكتب مساعدة يوفر منصة دعم عملاء قائمة على البريد الإلكتروني، وأداة قاعدة معرفية، وويدجت بحث/اتصال قابل للتضمين لمحترفي خدمة العملاء.

> **بيانات الاعتماد**
>
> يمكنك العثور على معلومات المصادقة لهذه العقدة [هنا](/integrations/builtin/credentials/helpscout.md).

> **أمثلة وقوالب**
>
> لأمثلة الاستخدام والقوالب لمساعدتك على البدء، ارجع إلى صفحة [تكاملات Help Scout Trigger](https://n8n.io/integrations/helpscout-trigger/) الخاصة بـ n8n.
