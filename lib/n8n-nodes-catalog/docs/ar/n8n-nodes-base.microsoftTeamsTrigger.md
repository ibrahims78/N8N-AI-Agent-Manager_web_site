# عقدة مُحفِّز Microsoft Teams

استخدم عقدة مُحفِّز Microsoft Teams للاستجابة للأحداث في [Microsoft Teams](https://www.microsoft.com/en-us/microsoft-teams/group-chat-software) ودمج Microsoft Teams مع التطبيقات الأخرى.

في هذه الصفحة، ستجد قائمة بالأحداث التي يمكن لعقدة مُحفِّز Microsoft Teams الاستجابة لها وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> يمكنك العثور على معلومات المصادقة لهذه العقدة [هنا](/integrations/builtin/credentials/microsoft.md).

> **دعم السحابة الحكومية**
>
> إذا كنت تستخدم مستأجر سحابة حكومية (حكومة الولايات المتحدة، وزارة الدفاع الأمريكية، أو الصين)، فتأكد من تحديد **عنوان URL الأساسي لواجهة برمجة تطبيقات Microsoft Graph** المناسب في إعدادات بيانات اعتماد Microsoft الخاصة بك.
