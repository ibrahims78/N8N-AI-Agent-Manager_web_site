# عقدة Azure Storage

تدعم عقدة Azure Storage (التخزين السحابي من مايكروسوفت) مجموعة واسعة من الميزات المضمنة، والتي تشمل إنشاء الكائنات الثنائية كبيرة الحجم (blobs) والحاويات، والحصول عليها، وحذفها. استخدم هذه العقدة لأتمتة العمل داخل خدمة Azure Storage أو لدمجها مع خدمات أخرى في سير العمل الخاص بك.

في هذه الصفحة، ستجد قائمة بالعمليات التي تدعمها عقدة Azure Storage، وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> يمكنك العثور على معلومات المصادقة لهذه العقدة [هنا](/integrations/builtin/credentials/azurestorage.md).

## العمليات

*   **الكائن الثنائي كبير الحجم (Blob)**
    *   **إنشاء كائن ثنائي كبير الحجم**: إنشاء كائن ثنائي كبير الحجم جديد أو استبدال كائن موجود.
    *   **حذف كائن ثنائي كبير الحجم**: حذف كائن ثنائي كبير الحجم موجود.
    *   **الحصول على كائن ثنائي كبير الحجم**: استرداد البيانات لكائن ثنائي كبير الحجم محدد.
    *   **الحصول على العديد من الكائنات الثنائية كبيرة الحجم**: استرداد قائمة بالكائنات الثنائية كبيرة الحجم.
*   **الحاوية (Container)**
    *   **إنشاء حاوية**: إنشاء حاوية جديدة.
    *   **حذف حاوية**: حذف حاوية موجودة.
    *   **الحصول على حاوية**: استرداد البيانات لحاوية محددة.
    *   **الحصول على العديد من الحاويات**: استرداد قائمة بالحاويات.

## ماذا تفعل إذا كانت العملية المطلوبة غير مدعومة

إذا كانت هذه العقدة لا تدعم العملية التي ترغب في القيام بها، يمكنك استخدام [عقدة HTTP Request](/integrations/builtin/core-nodes/n8n-nodes-base.httprequest/index.md) لاستدعاء واجهة برمجة التطبيقات (API) الخاصة بالخدمة.

يمكنك استخدام بيانات الاعتماد التي أنشأتها لهذه الخدمة في عقدة HTTP Request:

1.  في عقدة HTTP Request، حدد **Authentication** > **Predefined Credential Type**.
2.  حدد الخدمة التي تريد الاتصال بها.
3.  حدد بيانات الاعتماد الخاصة بك.

ارجع إلى [عمليات API المخصصة](/integrations/custom-operations.md) لمزيد من المعلومات.
