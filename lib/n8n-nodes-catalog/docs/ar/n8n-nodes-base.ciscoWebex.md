# عقدة Webex by Cisco

استخدم عقدة Webex by Cisco لأتمتة العمل في Webex، ودمج Webex مع التطبيقات الأخرى. تدعم n8n بشكل مدمج مجموعة واسعة من ميزات Webex، بما في ذلك إنشاء الاجتماعات والرسائل، واستردادها، وتحديثها، وحذفها.

ستجد في هذه الصفحة قائمة بالعمليات التي تدعمها عقدة Webex وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد Webex](/integrations/builtin/credentials/ciscowebex.md) للحصول على إرشادات حول إعداد المصادقة.
> **أمثلة وقوالب**
>
> للحصول على أمثلة استخدام وقوالب لمساعدتك على البدء، ألقِ نظرة على قائمة [تكاملات Webex](https://n8n.io/integrations/webex-by-cisco/) الخاصة بـ n8n.

## العمليات

* الاجتماع
    * إنشاء
    * حذف
    * استرداد
    * استرداد الكل
    * تحديث
* الرسالة
    * إنشاء

## ماذا تفعل إذا كانت عمليتك غير مدعومة

إذا كانت هذه العقدة لا تدعم العملية التي ترغب في تنفيذها، يمكنك استخدام [عقدة HTTP Request](/integrations/builtin/core-nodes/n8n-nodes-base.httprequest/index.md) لاستدعاء واجهة برمجة تطبيقات الخدمة.

يمكنك استخدام بيانات الاعتماد التي أنشأتها لهذه الخدمة في عقدة HTTP Request:

1. في عقدة HTTP Request، حدد **المصادقة** > **نوع بيانات الاعتماد المُحددة مسبقًا**.
2. حدد الخدمة التي ترغب في الاتصال بها.
3. حدد بيانات الاعتماد الخاصة بك.

ارجع إلى [عمليات API المخصصة](/integrations/custom-operations.md) للمزيد من المعلومات.
