# عقدة Harvest

استخدم عقدة Harvest لأتمتة سير العمل في Harvest، ودمج Harvest مع التطبيقات الأخرى. تدعم n8n بشكل مدمج مجموعة واسعة من ميزات Harvest، بما في ذلك إنشاء العملاء وجهات الاتصال والفواتير والمهام والمصروفات والمستخدمين والمشاريع وتحديثها وحذفها والحصول على بياناتها.

تجد في هذه الصفحة قائمة بالعمليات التي تدعمها عقدة Harvest وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد Harvest](/integrations/builtin/credentials/harvest.md) للحصول على إرشادات حول إعداد المصادقة.

## ماذا تفعل إذا كانت عمليتك غير مدعومة

إذا كانت هذه العقدة لا تدعم العملية التي ترغب في تنفيذها، يمكنك استخدام [عقدة HTTP Request](/integrations/builtin/core-nodes/n8n-nodes-base.httprequest/index.md) لاستدعاء واجهة برمجة تطبيقات (API) الخدمة.

يمكنك استخدام بيانات الاعتماد التي أنشأتها لهذه الخدمة في عقدة HTTP Request:

1.  في عقدة HTTP Request، حدد **المصادقة** > **نوع بيانات الاعتماد المُحددة مسبقًا**.
2.  حدد الخدمة التي ترغب في الاتصال بها.
3.  حدد بيانات الاعتماد الخاصة بك.

ارجع إلى [عمليات API المخصصة](/integrations/custom-operations.md) لمزيد من المعلومات.
