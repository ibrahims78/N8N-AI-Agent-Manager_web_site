# عقدة Iterable (منصة تسويق عبر البريد الإلكتروني والهاتف المحمول)

استخدم عقدة Iterable لأتمتة سير العمل في Iterable، ودمج Iterable مع تطبيقات أخرى. تدعم n8n بشكل مدمج مجموعة واسعة من ميزات Iterable، بما في ذلك إنشاء المستخدمين، وتسجيل الإجراءات التي يقوم بها المستخدمون، وإضافة المستخدمين إلى القوائم وإزالتهم منها.

في هذه الصفحة، ستجد قائمة بالعمليات التي تدعمها عقدة Iterable وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد Iterable](/integrations/builtin/credentials/iterable.md) للحصول على إرشادات حول إعداد المصادقة.

## العمليات

*   الحدث
    *   تسجيل الإجراءات التي يقوم بها المستخدم.
*   المستخدم
    *   إنشاء/تحديث مستخدم
    *   حذف مستخدم
    *   الحصول على مستخدم
*   قائمة المستخدمين
    *   إضافة مستخدم إلى قائمة
    *   إزالة مستخدم من قائمة
