# عقدة مُحفِّز WooCommerce

[WooCommerce](https://woocommerce.com/) (ووكومرس) هي إضافة تجارة إلكترونية مفتوحة المصدر وقابلة للتخصيص لـ ووردبريس.

> **بيانات الاعتماد**
>
> يمكن العثور على معلومات المصادقة لهذه العقدة [هنا](/integrations/builtin/credentials/woocommerce.md).

> **أمثلة وقوالب**
>
> لأمثلة الاستخدام والقوالب لمساعدتك على البدء، ارجع إلى صفحة [تكاملات مُحفِّز ووكومرس](https://n8n.io/integrations/woocommerce-trigger/) الخاصة بـ n8n.
