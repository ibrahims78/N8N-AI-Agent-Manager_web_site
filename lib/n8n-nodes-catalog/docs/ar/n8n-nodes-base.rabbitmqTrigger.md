# العقدة المُحفِّزة RabbitMQ

[RabbitMQ](https://www.rabbitmq.com) هو وسيط رسائل مفتوح المصدر يقبل الرسائل ويعيد توجيهها.

> **بيانات الاعتماد**
>
> يمكن العثور على معلومات المصادقة لهذه العقدة [هنا](/integrations/builtin/credentials/rabbitmq.md).

> **أمثلة وقوالب**
>
> للاطلاع على أمثلة الاستخدام والقوالب لمساعدتك على البدء، ارجع إلى صفحة [تكاملات المُحفِّز Rabbit MQ](https://n8n.io/integrations/rabbitmq-trigger/) الخاصة بـ n8n.

## الموارد ذات الصلة

توفر n8n عقدة تطبيق لـ RabbitMQ. يمكن العثور على وثائق العقدة [هنا](/integrations/builtin/app-nodes/n8n-nodes-base.rabbitmq.md).

يمكنك عرض [أمثلة سير العمل والمحتوى ذي الصلة](https://n8n.io/integrations/rabbitmq-trigger/) على موقع n8n الإلكتروني.
