# عقدة PostHog (أداة تحليل بيانات)

استخدم عقدة PostHog لأتمتة سير العمل في PostHog، ودمج PostHog مع تطبيقات أخرى. تدعم n8n بشكل مدمج مجموعة واسعة من ميزات PostHog، بما في ذلك إنشاء الأسماء المستعارة (aliases)، والأحداث (events)، والهوية (identity)، بالإضافة إلى تتبع الصفحات.

في هذه الصفحة، ستجد قائمة بالعمليات التي تدعمها عقدة PostHog وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد PostHog](/integrations/builtin/credentials/posthog.md) للحصول على إرشادات حول إعداد المصادقة.

## العمليات

* الاسم المستعار (Alias)
    * إنشاء اسم مستعار
* الحدث (Event)
    * إنشاء حدث
* الهوية (Identity)
    * إنشاء
* التتبع (Track)
    * تتبع صفحة
    * تتبع شاشة
