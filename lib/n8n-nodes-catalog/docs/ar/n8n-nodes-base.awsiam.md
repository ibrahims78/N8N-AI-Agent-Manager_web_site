# عقدة AWS IAM

استخدم عقدة AWS IAM لأتمتة سير العمل في AWS Identity and Access Management (IAM) ودمج AWS IAM مع التطبيقات الأخرى. تدعم n8n بشكل مدمج مجموعة واسعة من ميزات AWS IAM، والتي تشمل إنشاء المستخدمين والمجموعات وتحديثهم واسترجاعهم وحذفهم، بالإضافة إلى إدارة عضوية المجموعات.

في هذه الصفحة، ستجد قائمة بالعمليات التي تدعمها عقدة AWS IAM، وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> يمكنك العثور على معلومات المصادقة لهذه العقدة [هنا](/integrations/builtin/credentials/aws.md).

## العمليات

*   **المستخدم**:
    *   **إضافة إلى مجموعة**: إضافة مستخدم موجود إلى مجموعة.
    *   **إنشاء**: إنشاء مستخدم جديد.
    *   **حذف**: حذف مستخدم.
    *   **استرجاع**: استرجاع مستخدم.
    *   **استرجاع متعدد**: استرجاع قائمة بالمستخدمين.
    *   **إزالة من مجموعة**: إزالة مستخدم من مجموعة.
    *   **تحديث**: تحديث مستخدم موجود.
*   **المجموعة**:
    *   **إنشاء**: إنشاء مجموعة جديدة.
    *   **حذف**: حذف مجموعة.
    *   **استرجاع**: استرجاع مجموعة.
    *   **استرجاع متعدد**: استرجاع قائمة بالمجموعات.
    *   **تحديث**: تحديث مجموعة موجودة.

## ماذا تفعل إذا كانت عمليتك غير مدعومة

إذا كانت هذه العقدة لا تدعم العملية التي ترغب في تنفيذها، يمكنك استخدام [عقدة طلب HTTP](/integrations/builtin/core-nodes/n8n-nodes-base.httprequest/index.md) لاستدعاء واجهة برمجة تطبيقات الخدمة.

يمكنك استخدام بيانات الاعتماد التي أنشأتها لهذه الخدمة في عقدة طلب HTTP:

1.  في عقدة طلب HTTP، حدد **المصادقة** > **نوع بيانات الاعتماد المُحددة مسبقًا**.
2.  حدد الخدمة التي ترغب في الاتصال بها.
3.  حدد بيانات الاعتماد الخاصة بك.

ارجع إلى [عمليات API المخصصة](/integrations/custom-operations.md) لمزيد من المعلومات.
