# عقدة Mailjet

استخدم عقدة Mailjet (ميل جيت) لأتمتة سير العمل في Mailjet، ودمج Mailjet مع تطبيقات أخرى. تدعم n8n بشكل مدمج مجموعة واسعة من ميزات Mailjet، بما في ذلك إرسال رسائل البريد الإلكتروني والرسائل النصية القصيرة (SMS).

في هذه الصفحة، ستجد قائمة بالعمليات التي تدعمها عقدة Mailjet وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد Mailjet](/integrations/builtin/credentials/mailjet.md) للحصول على إرشادات حول إعداد المصادقة.

## العمليات

*   البريد الإلكتروني
    *   إرسال بريد إلكتروني
    *   إرسال قالب بريد إلكتروني
*   الرسائل النصية القصيرة (SMS)
    *   إرسال رسالة نصية قصيرة (SMS)

## ماذا تفعل إذا كانت عمليتك غير مدعومة

إذا كانت هذه العقدة لا تدعم العملية التي ترغب في تنفيذها، يمكنك استخدام [عقدة طلب HTTP (HTTP Request)](/integrations/builtin/core-nodes/n8n-nodes-base.httprequest/index.md) لاستدعاء واجهة برمجة تطبيقات (API) الخدمة.

يمكنك استخدام بيانات الاعتماد التي أنشأتها لهذه الخدمة في عقدة طلب HTTP (HTTP Request):

1.  في عقدة طلب HTTP (HTTP Request)، حدد **المصادقة** > **نوع بيانات الاعتماد المُحددة مسبقًا**.
2.  حدد الخدمة التي ترغب في الاتصال بها.
3.  حدد بيانات الاعتماد الخاصة بك.

ارجع إلى [عمليات API المخصصة](/integrations/custom-operations.md) لمزيد من المعلومات.
