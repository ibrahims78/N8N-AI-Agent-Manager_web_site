# عقدة X (تويتر سابقًا)

استخدم عقدة X لأتمتة سير العمل في X ودمج X مع التطبيقات الأخرى. تدعم n8n بشكل مدمج مجموعة واسعة من ميزات X، بما في ذلك إنشاء الرسائل المباشرة وحذف التغريدات، والبحث عنها، والإعجاب بها، وإعادة تغريدها.

في هذه الصفحة، ستجد قائمة بالعمليات التي تدعمها عقدة X وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد X](/integrations/builtin/credentials/twitter.md) للحصول على إرشادات حول إعداد المصادقة.

> **يمكن استخدام هذه العقدة كأداة ذكاء اصطناعي**
>
> يمكن استخدام هذه العقدة لتعزيز قدرات وكيل الذكاء الاصطناعي. عند استخدامها بهذه الطريقة، يمكن تعيين العديد من المعاملات تلقائيًا، أو بمعلومات موجهة بواسطة الذكاء الاصطناعي - اكتشف المزيد في [توثيق معاملات أداة الذكاء الاصطناعي](/advanced-ai/examples/using-the-fromai-function.md).
