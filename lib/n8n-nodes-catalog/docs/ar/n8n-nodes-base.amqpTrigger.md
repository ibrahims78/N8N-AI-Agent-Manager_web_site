# عقدة مُحفِّز AMQP

[AMQP](https://www.amqp.org/) هو بروتوكول طبقة تطبيق مفتوح المعايير لبرمجيات الوسيط الموجهة بالرسائل. تتمثل الميزات الأساسية لـ AMQP في توجيه الرسائل، والاصطفاف، والتوجيه، والموثوقية، والأمان. تدعم هذه العقدة وسطاء الرسائل المتوافقين مع AMQP 1.0.

> **بيانات الاعتماد**
>
> يمكنك العثور على معلومات المصادقة لهذه العقدة [هنا](/integrations/builtin/credentials/amqp.md).

> **أمثلة وقوالب**
>
> للاطلاع على أمثلة الاستخدام والقوالب لمساعدتك على البدء، ارجع إلى صفحة [تكاملات AMQP](https://n8n.io/integrations/amqp-trigger/) الخاصة بـ n8n.
