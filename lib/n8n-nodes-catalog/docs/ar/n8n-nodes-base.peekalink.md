# العقدة Peekalink

استخدم العقدة Peekalink (Peekalink node) لأتمتة سير العمل في Peekalink، ودمج Peekalink مع التطبيقات الأخرى. تدعم n8n فحص الروابط ومراجعتها باستخدام Peekalink.

ستجد في هذه الصفحة قائمة بالعمليات التي تدعمها العقدة Peekalink وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد Peekalink](/integrations/builtin/credentials/peekalink.md) للحصول على إرشادات حول إعداد المصادقة.

## العمليات

*   التحقق مما إذا كانت المعاينة متاحة لرابط معين
*   إرجاع معاينة لرابط
