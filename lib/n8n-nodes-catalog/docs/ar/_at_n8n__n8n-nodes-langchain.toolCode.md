# العقدة Custom Code Tool

استخدم العقدة Custom Code Tool لكتابة الكود الذي يمكن لـ [وكيل](/glossary.md#ai-agent) تشغيله.

في هذه الصفحة، ستجد معاملات العقدة للعقدة Custom Code Tool وروابط لموارد إضافية.

> **حل المعاملات في العقد الفرعية**
>
> تتصرف العقد الفرعية بشكل مختلف عن العقد الأخرى عند معالجة عناصر متعددة باستخدام تعبير.
>
> تأخذ معظم العقد، بما في ذلك العقد الجذرية، أي عدد من العناصر كمدخلات، وتعالج هذه العناصر، وتُخرج النتائج. يمكنك استخدام التعبيرات للإشارة إلى عناصر المدخلات، وتحل العقدة التعبير لكل عنصر على حدة. على سبيل المثال، بالنظر إلى مدخلات من خمس قيم `name`، يحل التعبير `` إلى كل اسم على حدة.
>
> في العقد الفرعية، يحل التعبير دائمًا إلى العنصر الأول. على سبيل المثال، بالنظر إلى مدخلات من خمس قيم `name`، يحل التعبير `` دائمًا إلى الاسم الأول.

## معاملات العقدة

### الوصف

امنح الكود المخصص الخاص بك وصفًا. يخبر هذا الوكيل متى يستخدم هذه الأداة. على سبيل المثال:

> استدعِ هذه الأداة للحصول على لون عشوائي. يجب أن تكون المدخلات سلسلة نصية تحتوي على أسماء ألوان مفصولة بفاصلة لاستبعادها.

### اللغة

يمكنك استخدام JavaScript أو Python.

### مربع JavaScript / Python

اكتب الكود هنا.

يمكنك الوصول إلى مدخلات الأداة باستخدام `query`. على سبيل المثال، لأخذ سلسلة المدخلات وتحويلها إلى أحرف صغيرة:

```js
let myString = query;
return myString.toLowerCase();
```
