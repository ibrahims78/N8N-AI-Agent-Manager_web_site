# عقدة Flow

استخدم عقدة Flow لأتمتة سير العمل في Flow، ودمج Flow مع تطبيقات أخرى. تدعم n8n مجموعة واسعة من ميزات Flow المدمجة، بما في ذلك إنشاء المهام وتحديثها والحصول عليها.

في هذه الصفحة، ستجد قائمة بالعمليات التي تدعمها عقدة Flow وروابط لموارد إضافية.

> **بيانات الاعتماد**
>
> ارجع إلى [بيانات اعتماد Flow](/integrations/builtin/credentials/flow.md) للحصول على إرشادات حول إعداد المصادقة.

## العمليات

*   المهمة
    *   إنشاء مهمة جديدة
    *   تحديث مهمة
    *   الحصول على مهمة
    *   الحصول على جميع المهام
