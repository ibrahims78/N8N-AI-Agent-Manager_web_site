# ماركداون

تحوّل عقدة ماركداون بين تنسيقات ماركداون و HTML.

## العمليات

عمليات هذه العقدة هي **الأنماط**:

*   **ماركداون إلى HTML**: استخدم هذا النمط للتحويل من ماركداون إلى HTML.
*   **HTML إلى ماركداون**: استخدم هذا النمط للتحويل من HTML إلى ماركداون.

## معاملات العقدة

*   **HTML** أو **Markdown**: أدخل البيانات التي ترغب في تحويلها. يتغير اسم الحقل بناءً على **النمط** الذي تحدده.
*   **Destination Key**: أدخل الحقل الذي ترغب في وضع المخرجات فيه. حدد الحقول المتداخلة باستخدام النقاط، على سبيل المثال `level1.level2.newKey`.

## خيارات العقدة

تعتمد **خيارات** العقدة على **النمط** المحدد.

> **اختبر الخيارات**
>
> تعتمد بعض الخيارات على بعضها البعض أو يمكن أن تتفاعل. نوصي باختبار الخيارات للتأكد من أن التأثيرات هي ما تريده.

The Markdown node converts between Markdown and HTML formats.

## Operations

This node's operations are **Modes**:

* **Markdown to HTML**: Use this mode to convert from Markdown to Markdown to HTML.
* **HTML to Markdown**: Use this mode to convert from HTML to HTML to Markdown.

## Node parameters

* **HTML** or **Markdown**: Enter the data you want to convert. The field name changes based on which **Mode** you select.
* **Destination Key**: Enter the field you want to put the output in. Specify nested fields using dots, for example `level1.level2.newKey`.

## Node options

The node's **Options** depend on the **Mode** selected.

> **Test out the options**
>
> Some of the options depend on each other or can interact. We recommend testing out options to confirm the effects are what you want.

### Markdown to HTML options

| Option | Description | Default |
| ------ | ----------- | ------- |

| **Complete HTML Document** | Whether to output a complete HTML document (enabled) or an HTML fragment (disabled). A complete HTML document includes the `<DOCTYPE HTML>` declaration, `<html>` and `<body>` tags, and the `<head>` element.  | Disabled |
| **Customized Header ID** | Whether to support custom heading IDs (enabled) or not (disabled). When enabled, you can add custom heading IDs using `{header ID here}` after the heading text. | Disabled |
| **Emoji Support** | Whether to support emojis (enabled) or not (disabled). | Disabled. |
| **Encode Emails** | Whether to transform ASCII character emails into their equivalent decimal entities (enabled) or not (disabled). | Enabled |

### خيارات تحويل HTML إلى Markdown

| الخيار | الوصف | الافتراضي |
| ------ | ----------- | ------- |
| **Bullet Marker** | تحديد الحرف المراد استخدامه للقوائم غير المرتبة. | * |
| **Code Block Fence** | تحديد الأحرف المراد استخدامها لكتل الكود. | ``` |
| **Emphasis Delimiter** | تحديد الحرف `<em>`. | _ |
| **Global Escape Pattern** | يتجاوز إعدادات الهروب الافتراضية للأحرف. قد ترغب في استخدام Text Replacement Pattern بدلاً من ذلك. | لا شيء |
| **Ignored Elements** | تجاهل عناصر HTML المحددة، وعناصرها الفرعية. | لا شيء |
| **Keep Images With Data** | ما إذا كان سيتم الاحتفاظ بالصور التي تحتوي على بيانات (مُمكّن) أم لا (معطل). يدعم الملفات حتى 1 ميجابايت. | معطل |
| **Line Start Escape Pattern** | يتجاوز إعدادات الهروب الافتراضية للأحرف. قد ترغب في استخدام Text Replacement Pattern بدلاً من ذلك. | لا شيء |
| **Max Consecutive New Lines** | رقم. تحديد الحد الأقصى لعدد الأسطر الجديدة المتتالية المسموح بها. | 3 |
| **Place URLs At The Bottom** | ما إذا كان سيتم وضع عناوين URL في أسفل الصفحة وتنسيقها باستخدام تعريفات مرجع الروابط (مُمكّن) أم لا (معطل). | معطل |
| **Strong Delimiter** | تحديد الأحرف لـ `<strong>`. | ** |
| **Style For Code Block** | تحديد نمط كتل الكود. الخيارات هي **Fence** و **Indented**. | Fence |
| **Text Replacement Pattern** | تحديد نمط استبدال النص باستخدام regex. | لا شيء |
| **Treat As Blocks** | تحديد عناصر HTML التي سيتم التعامل معها ككتل (إحاطتها بأسطر فارغة) | لا شيء |
